# 微信小程序需求说明书：UbuntuNews & AI 助手

## 1. 项目背景
基于现有的公众号内容（Ubuntu、Linux 资讯、AI 本地部署教程），开发一款服务于 Linux 爱好者和 AI 开发者的小程序。旨在提供更便捷的文档检索、常用命令查询以及 AI 工具部署指南。

## 2. 目标用户
- Linux 初学者（安装系统、配置环境）
- 资深 Linux 用户（关注内核更新、新发行版资讯）
- AI 开发者/爱好者（关注本地大模型部署、DeepSeek/Qwen 等模型应用）

## 3. 功能架构

### 3.1 底部导航菜单 (TabBar)
1.  **资讯 (News)** - 首页
2.  **百科 (Wiki)** - 知识库/工具箱
3.  **AI 实验室 (AI Lab)** - AI 专题
4.  **我的 (Me)** - 个人中心

### 3.2 详细功能模块

#### 3.2.1 资讯 (News)
*   **精选 Banner**：展示最新的重磅新闻（如 Ubuntu 26.04 发布、DeepSeek 教程）。
*   **资讯流**：
    *   **分类**：全部、系统更新、开源周报、AI 动态、教程。
    *   **列表样式**：左文右图，包含发布时间、阅读量。
*   **搜索栏**：全局搜索文章内容。

#### 3.2.2 百科 (Wiki)
*   **常用命令速查**：
    *   Linux 基础命令 (ls, cd, grep...)
    *   Git 常用命令
    *   Vim 快捷键图谱
    *   Docker/K8s 速查
*   **发行版指南**：
    *   Ubuntu/Debian/Arch/Fedora 安装配置向导。
    *   常见问题 (FAQ) 解决（如：显卡驱动、双系统引导）。
*   **环境配置脚本**：提供常用开发环境（Python, Node.js, Go, Rust）的一键安装命令/脚本复制功能。

#### 3.2.3 AI 实验室 (AI Lab)
*   **模型部署向导**：
    *   DeepSeek-R1 本地部署（Ollama/vLLM）。
    *   Qwen/Llama 模型量化与运行。
*   **AI 编程工具**：
    *   Cursor、Windsurf、Trae 等 AI 编辑器使用技巧。
    *   终端 AI 工具推荐 (Atuin, Warp)。
*   **Prompt 库**：精选 Linux 运维、代码生成的提示词模版。

#### 3.2.4 我的 (Me)
*   **收藏夹**：收藏的文章和教程。
*   **阅读历史**：最近浏览的内容。
*   **关于我们**：公众号二维码、GitHub 仓库链接。
*   **反馈建议**：用户提交内容需求或 Bug 反馈。

## 4. 非功能需求
*   **界面风格**：极简、Geek 风（暗黑模式支持），主色调采用 Ubuntu 橙 (#E95420) 和 终端黑。
*   **性能**：文章加载速度快，代码块支持高亮且可一键复制。
*   **数据同步**：尝试与公众号文章数据打通（如通过 RSS 或手动录入）。

## 5. 扩展规划 (V2.0)
*   **积分系统**：阅读文章获取积分，兑换周边。
*   **社区讨论**：文章下方的评论区。
*   **在线终端**：简单的 SSH 连接工具或模拟终端练习。
