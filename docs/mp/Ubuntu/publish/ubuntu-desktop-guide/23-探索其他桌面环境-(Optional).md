# 23. 探索其他桌面环境 (Optional)

Ubuntu 默认使用 GNOME，但 Linux 的魅力在于“可以换”。你可能因为三种原因想换桌面环境：更像 Windows 的布局、更省资源、更可定制。别把它当成“必须折腾”，当成一次对工作流的选择就好。

## 1. KDE Plasma (Kubuntu)

*   **特点**: 极其华丽，定制性极强（你想改哪里就能改哪里），布局类 Windows。
*   **资源占用**: 现在的 KDE 优化极好，甚至比 GNOME 还省内存。
*   **安装**:
    ```bash
    sudo apt install kde-plasma-desktop
    ```
    *安装过程中会问你选择 gdm3 还是 sddm，选 sddm (KDE 的显示管理器)。*

## 2. XFCE (Xubuntu)

*   **特点**: 轻量级，稳定，复古。
*   **适用**: 老旧电脑，服务器，追求极致响应速度的用户。
*   **安装**:
    ```bash
    sudo apt install xfce4
    ```

## 3. 切换桌面环境

安装完新桌面后，不需要重装系统。
1.  **注销 (Log Out)** 当前用户。
2.  在登录界面，点击右下角（或用户名旁边）的齿轮图标。
3.  选择你想进入的桌面（如 "Plasma", "Xfce Session"）。
4.  输入密码登录。

## 4. Wayland vs X11

在登录界面的齿轮里，你可能还会看到 "Ubuntu" 和 "Ubuntu on Xorg"。
*   **Wayland**: 新一代显示协议，更安全，画面撕裂更少，触控板手势更顺滑。是 Ubuntu 的默认选择。
*   **X11 (Xorg)**: 老牌协议，兼容性最好（尤其是老显卡、录屏软件、远程控制）。

**如果遇到花屏、录屏黑屏、输入法不跟随等怪问题，尝试切换回 X11 试试。**

经验上：

- 日常桌面使用：Wayland 往往更顺滑
- 远程控制/录屏/老硬件：X11 更“兼容优先”

## 5. 彻底更换发行版？

如果你发现自己特别喜欢 KDE，下次重装时可以直接下载 **Kubuntu**；喜欢 XFCE 可以下载 **Xubuntu**。它们都是 Ubuntu 的官方变体，只是预装的桌面环境不同。

如果你还在“挑桌面阶段”，也可以不安装到硬盘：用 U 盘启动 Live 环境先体验 10 分钟，确定喜欢再决定是否折腾切换。
