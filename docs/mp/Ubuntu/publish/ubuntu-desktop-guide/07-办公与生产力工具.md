# 07. 办公与生产力工具

![封面图](../../imgs/ubuntu-desktop-guide-07-cover.svg)

“能不能办公”通常不是问题，真正的问题是：**文件格式兼容、字体、以及你所在团队的工具链**。这一章按最常见的办公场景把坑填平。

## 1. 办公套件：LibreOffice vs WPS

### LibreOffice (预装)
*   **特点**: 完全开源，格式标准 (ODF)，兼容微软 Office 格式。
*   **体验**: 界面偏传统（类似 Office 2003），功能极其强大但上手需要适应。
*   **适用**: 纯文本处理，对格式兼容性要求不严苛的场景。

### WPS Office for Linux (强烈推荐)
*   **特点**: 金山官方出品，原生 Linux 版。
*   **体验**: 界面与 Windows 版几乎一致，对 `.docx`, `.xlsx`, `.pptx` 的兼容性极佳，包含云文档功能。
*   **安装**: 去 WPS 官网下载 `.deb` 包安装。
    *   *注意*: 安装后可能提示 "系统缺失字体"，只需按照上一篇教程安装 Windows 字体即可解决。

如果你经常需要“和 Windows 用户来回互改同一个 `.docx/.pptx`”，更建议 WPS；如果你更在意标准与可迁移性（比如长期归档、学术写作），LibreOffice 也很稳。

## 2. PDF 阅读与编辑

*   **Document Viewer (Evince)**: 系统自带，启动快，适合浏览。
*   **Master PDF Editor**: 功能强大的 PDF 编辑器，支持修改文本、签名、注释（免费版有水印，可购买注册）。
*   **WPS PDF**: WPS 套件自带，功能够用。

如果你只是需要简单处理（合并、拆分、旋转、压缩），很多需求用网页工具也能解决；但涉及隐私/合同，尽量用本地工具。

## 3. 截图与录屏

### GNOME 自带工具
按下 `PrtSc` 键，界面非常现代。
*   支持选区、窗口、全屏截图。
*   支持录制屏幕（右下角切换到录像模式）。

### Flameshot (火焰截图) - 强力推荐
如果你需要类似 QQ/微信截图那样强大的标注功能（画箭头、马赛克、文字、序号）。
```bash
sudo apt install flameshot
```
建议在设置中将 `PrtSc` 快捷键绑定到 `flameshot gui` 命令。

## 4. 即时通讯 (IM)

这曾是 Linux 的痛点，但现在已有很大改善。

*   **微信 (WeChat)**:
    *   **原生版**: 腾讯最近发布了原生 Linux 版微信，功能虽然基础（聊天、传文件），但够用且省电。
    *   **Wine/Deepin 版**: 功能更全（朋友圈、小程序），但资源占用稍高。
*   **QQ**:
    *   腾讯发布了基于 Electron 的新版 Linux QQ，界面现代化，功能几乎与 Windows 版同步（支持语音视频、截图、群管理）。
*   **Telegram / Discord / Slack**:
    *   这些国际化软件都有非常优秀的 Linux 原生客户端，直接在 App Center 下载即可。

小建议：IM 软件尽量优先用原生/官方渠道安装；Wine 能解决“能用”的问题，但更难解决“省电、稳定、不卡顿”。

## 5. 邮件客户端

*   **Thunderbird**: Mozilla 出品，系统预装。功能强大，支持插件，适合重度邮件用户。
*   **Mailspring**: 界面非常漂亮现代，支持追踪邮件是否被阅读。

通过这些工具的组合，Ubuntu 完全可以胜任日常的文案、表格、沟通等办公任务。最影响体验的往往不是软件本身，而是字体与格式兼容：这部分在上一章解决掉后，办公体验会顺很多。
