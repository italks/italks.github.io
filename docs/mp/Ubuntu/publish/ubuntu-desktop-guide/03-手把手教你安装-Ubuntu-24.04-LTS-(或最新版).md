# 03. 手把手教你安装 Ubuntu 24.04 LTS (或最新版)

这章的目标很明确：把 Ubuntu 装到真机上，并且装完能正常启动、能联网、驱动不翻车。

**重要提醒**：涉及分区就有风险。别挑战运气，先备份。

## 1. 进入 BIOS/UEFI 设置启动项

这是新手最容易卡住的地方，因为不同品牌的电脑进入 BIOS 的按键不同。

1.  **插入 U 盘**：将制作好的 Ubuntu 启动盘插入电脑。
2.  **重启电脑**：在开机出现品牌 Logo 时，疯狂按下启动菜单快捷键（通常是 F12, F11, F10, Esc 或 Del）。
    *   *联想 (Lenovo)*: F12 或 Fn+F12
    *   *戴尔 (Dell)*: F12
    *   *惠普 (HP)*: F9
    *   *华硕 (Asus)*: F8 或 Esc
    *   *宏碁 (Acer)*: F12
3.  **选择启动项**：在菜单中找到你的 U 盘（通常带有 UEFI: SanDisk, UEFI: Kingston 等字样），回车进入。
4.  **GRUB 菜单**：你会看到一个黑底白字的菜单，选择第一项 `Try or Install Ubuntu` 并回车。

如果你看到了同一个 U 盘出现两项（带 UEFI 和不带 UEFI）：优先选 **带 UEFI 的那一项**。

## 2. 体验与开始安装

系统加载完成后，你会进入一个临时的 Ubuntu 桌面。
*   **Try Ubuntu (试用)**：你可以在不安装的情况下体验系统，检查 Wi-Fi、声音、蓝牙是否正常工作。这是一个非常好的排错机会。
*   **Install Ubuntu (安装)**：点击桌面上的 "Install Ubuntu" 图标开始安装向导。

建议先用 “Try Ubuntu” 做 3 件事（5 分钟搞定）：

1. 连上 Wi‑Fi，看能不能正常上网
2. 插拔一次鼠标/键盘/蓝牙耳机，确认外设基本可用
3. 如果你是独显（尤其是 NVIDIA），看看分辨率是否正常、有没有明显花屏/卡死

## 3. 安装向导详解

### A. 语言与键盘布局
*   **Language**: 选择 "中文(简体)"，如果你想锻炼英语，选择 "English" 也可以（后续可以改）。
*   **Keyboard**: 通常默认 "English (US)" 即可。

### B. 连接网络
建议连接 Wi-Fi，这样安装程序可以自动下载更新和第三方驱动（如显卡、网卡驱动）。

### C. 更新和其他软件
*   **安装类型**：
    *   **Normal installation (正常安装)**：包含浏览器、办公软件、游戏、媒体播放器等。**推荐新手选择**。
    *   **Minimal installation (最小安装)**：只包含浏览器和基本工具。适合老手。
*   **其他选项**：
    *   勾选 "Install third-party software..." (安装图形硬件和 Wi-Fi 硬件的第三方软件...)。**强烈建议勾选**，这能省去后续很多驱动麻烦。

如果你想把系统“收拾得更干净”：也可以选最小安装，后面再按需装软件；但第一次装机，更建议正常安装，减少变量。

### D. 安装类型 (关键步骤！)
这里决定了你的硬盘如何分配。

*   **情况一：安装双系统 (Install Ubuntu alongside Windows)**
    *   如果安装程序检测到了 Windows，会显示这个选项。
    *   它会自动调整 Windows 分区的大小，腾出空间给 Ubuntu。
    *   你可以拖动滑块来分配两个系统的大小（建议给 Ubuntu 至少 50GB）。
    *   **这是最简单、最安全的双系统安装方式。**

*   **情况二：清除整个磁盘 (Erase disk and install Ubuntu)**
    *   **警告**：这会删除硬盘上的所有数据！
    *   适合单系统用户或新硬盘。

*   **情况三：其他选项 (Something else)**
    *   手动分区。适合高级用户。如果你一定要手动分区，推荐方案：
        *   `EFI` 分区：500MB (如果已有 Windows EFI 分区，则不需要新建，只需挂载)。
        *   `/` (根分区)：剩余所有空间，格式为 ext4。
        *   *(可选)* `/home`：独立分区，方便重装系统保留数据。
        *   *(可选)* `swap`：交换分区，大小等于内存或内存的 2 倍（现在的 Ubuntu 会自动使用 swap file，通常不需要独立分区）。

双系统用户常见的两个“坑”：

- **Windows 还开着 BitLocker/设备加密**：装系统本身没问题，但后续引导/分区调整可能触发恢复密钥验证；装之前把密钥备好
- **Windows 快速启动没关**：容易导致 Ubuntu 挂载不了 Windows 分区，或者出现“分区被占用”

### E. 设置用户
*   **Your name**: 你的名字。
*   **Computer's name**: 电脑在网络中显示的名字。
*   **Username**: 登录用户名（只能小写字母）。
*   **Password**: 登录密码（**一定要记住！** Linux 做任何系统级操作都需要它）。

## 4. 等待安装与重启

点击 "Install Now" 后，系统开始复制文件并配置。这通常需要 10-20 分钟。
安装完成后，会提示 "Installation Complete"。

1.  点击 "Restart Now"。
2.  屏幕会提示 "Please remove the installation medium, then press ENTER"。
3.  拔掉 U 盘，按回车。

恭喜你！第一次进入系统后，先别急着装软件，建议按这个顺序做“开机三件套”：

1. 打开“软件更新器/更新”（或稍后在终端用 APT）把系统更新到最新补丁
2. 打开“软件与更新”检查额外驱动（尤其是 NVIDIA/无线网卡）
3. 重启一次，确认引导菜单里 Ubuntu/Windows 都能正常进入（双系统尤其重要）
