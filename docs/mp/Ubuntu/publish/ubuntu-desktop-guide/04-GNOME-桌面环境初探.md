# 04. GNOME 桌面环境初探

GNOME 的核心思路不是“把窗口最小化塞回任务栏”，而是：**用搜索打开应用，用工作区分组任务，用快捷键快速切换**。一开始会有点别扭，但只要把这套节奏用顺，效率会很高。

## 1. 核心界面元素概览

### A. 顶栏 (Top Bar)
位于屏幕最顶部的黑条。
*   **左侧**: "Activities"（活动）按钮，或左上角的圆点/Logo。点击它（或按 `Super/Win` 键）进入活动概览。
*   **中间**: 日期和时间。点击可查看通知中心和日历。
*   **右侧**: 系统状态菜单。包含网络、音量、电池、电源选项等。

### B. 活动概览 (Activities Overview)
按下键盘上的 `Super` 键（即 Windows 键）触发。这是 GNOME 的灵魂。
*   **搜索栏**: 在这里输入任何内容——应用名称、文件名、设置项，甚至是简单的计算题。
*   **Dock (左侧或底部)**: 常用应用的快捷栏。
*   **工作区 (Workspaces)**: 位于屏幕上方或右侧的缩略图。你可以把不同的窗口拖到不同的工作区，实现多任务分组（例如：工作区 1 放浏览器，工作区 2 放代码编辑器）。

## 2. 窗口管理与多任务

GNOME 鼓励使用动态工作区和快捷键来管理窗口，而不是最小化。

*   **最大化/还原**: 双击标题栏，或拖动窗口到屏幕顶部。
*   **半屏分屏**: 拖动窗口到屏幕左边缘或右边缘，窗口会自动占据一半屏幕。
*   **切换应用**: `Alt + Tab`。
*   **切换同一个应用的不同窗口**: `Alt + ~` (Tab 上面的键)。
*   **关闭窗口**: `Alt + F4` 或 `Ctrl + Q` (部分应用)。

如果你经常在多个任务之间切换（写文档、查资料、聊天、写代码），建议养成习惯：

- 一个工作区只做一件事（比如“写作”工作区里只放文档和资料）
- 切换任务时切工作区，而不是在一堆窗口里翻

## 3. 系统设置详解

点击顶栏右侧的系统菜单 -> "Settings" (齿轮图标)，这里是控制系统的中心。

### A. 显示器 (Displays)
*   **Resolution**: 分辨率设置。
*   **Scale**: 缩放比例。高分屏（HiDPI）用户建议开启 200% 或启用 "Fractional Scaling" (分数缩放) 来获得最佳视觉体验。
*   **Night Light**: 夜间模式，过滤蓝光保护眼睛。

### B. 外观 (Appearance)
*   **Style**: 切换浅色/深色模式 (Light/Dark)。
*   **Color**: 选择强调色。
*   **Dock**: 设置 Dock 栏的位置（左/底/右）、大小以及是否自动隐藏。

### C. 鼠标与触控板 (Mouse & Touchpad)
*   **Touchpad**: 笔记本用户强烈建议开启 "Tap to click" (轻触点击)。
*   **Scroll Direction**: 默认是 "Natural Scrolling" (自然滚动，类似手机)，如果不习惯可以关闭以恢复传统滚动方向。

### D. 电源 (Power)
*   **Power Mode**: 性能模式 (Performance) / 平衡 (Balanced) / 省电 (Power Saver)。
*   **Screen Blank**: 设置屏幕自动关闭的时间。

## 4. 快捷键是效率的关键

先背这几个就够用：

| 目的 | 快捷键 |
| --- | --- |
| 打开概览/搜索 | `Super` |
| 打开应用列表 | `Super + A` |
| 锁屏 | `Super + L` |
| 切应用 | `Alt + Tab` |
| 同一应用切窗口 | `Alt + ~` |
| 分屏 | `Super + ← / →` |
| 最大化/还原 | `Super + ↑ / ↓` |
| 截图/录屏 | `PrtSc` |

熟悉了这些，你就已经迈出了掌控 Ubuntu 的第一步。
