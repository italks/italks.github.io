# Ubuntu 桌面实战：从装机到进阶（系列大纲）

> **摘要**：本文是 Ubuntu 桌面实战系列的导读大纲，为想将 Ubuntu 作为主力桌面或需要 Linux 开发能力的读者提供学习路径。涵盖从虚拟机体验、双系统安装到开发环境搭建的多种场景，并说明了教程的约定与习惯，帮助读者快速上手并避开常见误区。

这套文章写给两类人：

- 想把 Ubuntu 当主力桌面用的人（办公、学习、轻开发）
- 需要 Linux 能力的人（命令行、开发环境、系统管理）

默认以 Ubuntu 24.04 LTS 为例；Deepin / UOS / 麒麟等基于 Debian/Ubuntu 的系统，很多章节可以直接照搬思路与命令。

## 怎么读更省时间

- **只想先体验**：01 → 02（虚拟机/WSL 方案）→ 04 → 05 → 06
- **准备装双系统**：01 → 02 → 03 → 04 → 05 → 06 → 09（权限与 sudo 很快用得上）
- **直接做开发机**：01 → 03 → 05 → 08 → 11 → 13 → 14 → 15

## 约定与习惯

- 代码块里的命令默认在终端执行；以 `sudo` 开头的命令会要求输入密码
- 能用图形界面解决的，就先用图形界面；需要批量/自动化时再回到命令行
- 每章都会尽量给出“这么做的原因”和“常见坑怎么避开”

---

## 第一阶段：装得上、装得稳（入门篇）

**目标**：把系统装好、能启动、关键硬件能用、遇到引导问题也不慌。

1. **01. 为什么要选择 Linux/Ubuntu？**  
   把优势与限制都说清：适合谁、不适合谁，以及国产发行版的共通点。
2. **02. 准备工作与安装方式选择**  
   把虚拟机 / WSL2 / 双系统 / 单系统放到一个表里对比，并给出选择建议与备份清单。
3. **03. 手把手教你安装 Ubuntu 24.04 LTS（或最新版）**  
   从 UEFI/安全启动到分区策略，再到安装后第一轮检查与排障入口。

## 第二阶段：桌面像桌面一样好用（基础篇）

**目标**：熟悉 GNOME 的工作流，完成日常办公、上网、输入法与软件安装。

4. **04. GNOME 桌面环境初探**  
   用“工作区 + 搜索 + 快捷键”建立高效用法，而不是把 GNOME 当 Windows 复刻。
5. **05. 软件安装与管理：告别 .exe**  
   讲清 DEB / Snap / Flatpak 的差别、边界与选择心法，避免“装是装上了但系统越来越乱”。
6. **06. 中文环境配置与输入法**  
   把中文显示、输入法框架、字体与常见兼容问题一次解决。
7. **07. 办公与生产力工具**  
   把“能不能用”变成“用起来顺手”：文档、PDF、截图录屏、通讯与替代方案。

## 第三阶段：终端不是黑魔法（进阶篇）

**目标**：掌握足够的命令行能力：定位文件、改配置、装软件、看日志、懂权限。

8. **08. 终端（Terminal）的第一步**  
   从路径与文件操作入手，把最常用的命令练到“肌肉记忆”。
9. **09. 文件权限与用户管理**  
   讲清 `rwx`、用户/用户组与 `sudo`，顺带解释为什么 Linux 更难“中毒”。
10. **10. 文本编辑之争：Nano 与 Vim 入门**  
   以“改配置文件不翻车”为目标，给出最短上手路径与常用按键清单。
11. **11. 命令行下的软件管理（APT）**  
   把更新、依赖、源与清理说透，避免“越用越慢、越装越乱”。

## 第四阶段：把它变成趁手的开发机（开发者篇）

**目标**：高效、可复现、可迁移：换台机器也能快速恢复工作环境。

12. **12. 终端美化与效率工具**  
   Zsh/Tmux 的“必要配置”，不追求花哨，追求稳定与省时间。
13. **13. Git 版本控制环境**  
   从 SSH、签名到常用工作流，让 Git 从“能用”变成“放心用”。
14. **14. 搭建编程环境**  
   以 Python/Node/C/C++ 为例，讲清版本管理、隔离与常见坑。
15. **15. 容器化技术：Docker**  
   从安装与权限到 Compose 上手，给出最小可复用的部署范式。

## 第五阶段：系统管理与故障排查（精通篇）

**目标**：知道系统为什么这样跑；遇到问题能定位、能恢复、能留证据。

16. **16. 进程管理与系统监控**  
   从 CPU/内存/IO 视角理解卡顿与异常，掌握查因与止损手段。
17. **17. 磁盘管理与文件系统**  
   分区、挂载、`fstab`、空间分析与 LVM 基础，把“磁盘”从黑箱变成工具。
18. **18. 网络配置与远程访问**  
   DNS/路由/端口、SSH 与文件传输，配合 UFW 做到“能连且安全”。
19. **19. Systemd 服务管理**  
   用 systemd 管住服务与日志，构建可维护的后台任务。
20. **20. 备份与恢复策略**  
   给出可执行的 3-2-1 备份思路与 Timeshift/rsync 落地方案。

## 第六阶段：定制与扩展（扩展篇）

**目标**：用可控的方式“折腾”，而不是折腾到最后只能重装。

21. **21. 桌面美化（Ricing）**  
   先讲可逆操作与回滚，再讲主题、图标、扩展与 Dock。
22. **22. 运行 Windows 应用：Wine 与 Proton**  
   讲清适用边界与替代方案，避免在不适合的地方硬刚。
23. **23. 探索其他桌面环境（Optional）**  
   什么时候该换桌面环境，Wayland/X11 的现实差别与切换策略。
24. **24. 总结与进阶学习路线**  
   给出继续深入的方向：桌面、开发、运维、安全、内核，各自从哪里起步。
