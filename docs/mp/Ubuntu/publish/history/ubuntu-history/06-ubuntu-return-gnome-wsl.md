# Ubuntu 演进史（六）：妥协中的重生——放弃 Unity、重返 GNOME 与向微软和解（WSL）

在长达百年的西方管理学经典案例中，最考验一位舵手智慧的，往往不是他带领企业在风口上飞行的姿态，而是他能在巨大的沉没成本面前，用多快、多残忍的速度砍掉那个曾经视为心肝宝贝的战略支柱。

时间来到了 2017 年的春天。在这个时刻，那个曾经要用“大一统”战略横跨智能手机、平板和桌面的 Unity 8 及其底层的 Mir 显示架构生态，已经在持续近五年的数次跳票、生态枯竭中步入了绝境。无论是全球的终端开发者，还是那些曾狂热支持千万级众筹极客们，都在这套系统卡顿且没有任何主流应用（App）上架的泥潭中消磨尽了所有的耐心。由于长年在自研生态中闭门造车，Canonical 甚至疏远了全球最大的 Linux 社区的贡献力量体系。而庞大的底层开发人员的开支，依然在如无底洞般消耗着马克·沙特尔沃思的私产。

是继续抱着那不切实际的终端全平台梦想一同沉没，还是像一个冷酷的商人那样斩断情丝？

## 壮士断腕： Unity 帝国的瓦解与向开源社区的低头

2017 年 4 月 5 日。科技史应该铭记这一天。

马克·沙特尔沃思在 Ubuntu 社区的官方博客上发布了一篇极其沉痛但在字里行间充满了罕见清醒的公开告别信。他直接、且毫不掩饰地宣布了一个震撼业界的消息：Canonical 将立刻停止甚至解散所有关于智能手机、平板电脑大一统项目（Unity 8 与智能设备）的投资和开发；从同年的 Ubuntu 17.10 桌面版开始，操作系统将彻底抛弃 Unity 界面，从迷途重归到开源界最为庞大、最受工程师欢迎的主线——GNOME 桌面环境的怀抱中。

这是一次极具悲剧色彩的技术撤退。它不仅宣布了 Canonical 的大规模降本裁员，在本质上更宣判了曾风靡一时的“手机变身 PC（大一统）”这种狂妄的硬件融合尝试在当时的不可行性。马克坦承，那些自建生态的技术城墙虽然在工程上可能极其卓越，但在没有谷歌（Android）的资金链或者苹果（iOS）封闭霸权体系的滋养下，“集市”永远拼不过“垄断财团”的硬件重工业。他在信中那句极其苦涩的反思，揭露了跨界操作系统的最大死穴：“你在没有应用生态的地方苦苦维持创新，就像是去建造并不存在的未来火星殖民地。”

但是，这并非一次可耻的溃退，而是一次至关重要的断臂求生。

放弃了“造轮子”的包袱，Ubuntu 得以重新回归初心——将极度有限的顶尖工程师资源收缩聚拢在系统底层内核优化、云服务器维护环境以及那些能真正产生高价值企业服务的“刀刃”上。而就在这个自我阉割的节点，另一个在科技界有着百年宿仇的对象，却以一种所有人都无法置信的方式，向这只带伤的企鹅伸出了那双曾经沾满绞杀开源先烈鲜血的手。

## 化敌为友的魔幻史诗：从“毒瘤”到“微软爱 Linux”

如果你稍微回顾一下世纪之交时关于这头巨型企业微软的历史，你会发现前任掌门人鲍勃默曾咬牙切齿地将所有开源的 Linux 称为数字版权的“毒瘤”。但在史蒂夫·鲍尔默退居二线，当极其务实、且极具公有云（Azure）战略智慧的萨提亚·纳德拉（Satya Nadella）接手微软帝国时，整个科技圈的底层逻辑变天了。

纳德拉是一个极其清醒的现实工程派：既然无论你怎么绞杀，全世界数十亿极客以及上万家企业级客户都在用最脚投票的方式支持 Linux 作为唯一的开发编译底座环境。如果 Windows 再不能在这个云原生的狂欢时代与开源系统握手言和，那么微软那曾经高傲的“桌面神教”，将会不可避免地遭到整整一代开发者的群体遗弃。

2016 年到 2017 年期间，一场被极客们戏称为“世纪最不可思议和解”的大戏上演了。在微软那场充满“Microsoft Loves Linux（微软爱开源）”横幅的布道大会上，微软联合了曾经誓死方休的老对手 Canonical（即 Ubuntu 母公司），推出了一项改变了今天数亿程序员工作习惯的重大底层颠覆——**Windows Subsystem for Linux (WSL)**。

## WSL：借体还魂的终极外挂

WSL 的本质是什么？它不是过去那种需要模拟整套硬件层、并且消耗庞大内存的虚拟机（如 VMware）。在 WSL 的初期以及后续的 WSL 2 版本中，微软等于在其极度封闭的 Windows 内核底层旁边，甚至是不惜动用了最底层的 Hyper-V 硬件加速层，极其强悍且毫无保留地为运行一个原生的 Ubuntu 系统开凿出了一条超级专线。

你在自己那台用于打游戏、做 PPT、甚至带有华丽壁纸的 Windows 11 桌面上，只需在微软应用商店里敲击一下回车键，一个无需冗长安装、与 Windows 文件系统实现超深度互通（甚至共享硬盘盘符）、运行着全世界最新全栈开源组件的完整纯正 Ubuntu 就瞬间苏醒了。

从战略层面上看，这是 Ubuntu 历史上一次极其魔幻的“寄生路线”大捷。

马克和他的 Canonical 公司承认了自己在占据个人家用主机的惨败，但他却通过这种方式如同特洛伊木马一般，顺理成章地住进了微软的腹地大本营。这使得今天成千上万原本因为极其复杂的开发依赖配置（如 Docker、Node.js 原生模块、C++交叉编译等）而不得不痛苦折腾或去购买高昂苹果电脑（MacBook）的年轻一代后端工程师、甚至是数据科学家们，能够极其优雅地在 Windows 电脑上享受到极度原生的 Ubuntu 丝滑开发体验。

它没有打败那个不可一世的桌面宿敌，它只是冷静地与宿敌完成了极其罕见的深海融合。当 Ubuntu 以这种无比极客、也无比妥协的寄生方式夺取了全球最大桌面操作系统的这群大脑控制权时，它的战场，已经悄然完成了全方位的转移。
