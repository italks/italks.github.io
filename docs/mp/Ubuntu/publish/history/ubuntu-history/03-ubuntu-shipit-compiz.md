# Ubuntu 演进史（三）：寄送光盘（ShipIt）与流光溢彩的岁月——GNOME 2 霸权与 3D 桌面狂欢

从科技史的发展来看，任何伟大的软件不仅需要自身的代码精湛，更需要一条极其通畅的宣发与物理分发渠道。

在 21 世纪的头五年，尽管互联网泡沫已被吹爆并迎来重建，但在全球绝大多数发展中国家、乃至许多发达国家的偏远地区，家用宽带依然是极尽奢侈的存在。动辄数百兆、乃至一两张 CD 容量大小（约 700MB）的系统镜像文件，如果是通过极其不稳定的 56K 拨号网络下载，不仅需要耗费数十小时，更可能由于极其频繁的断线而功亏一篑。

在这条因为客观基础设施落后而形成的天然物理鸿沟面前，微软的 Windows 甚至更早期的 Linux 发行版都显得无可奈何，又或者只能依靠那些层层加价的盗版光盘贩子。然而，马克·沙特尔沃思却用他数亿美元的资产中极为微小的一环，硬生生地在这个鸿沟之上架起了一座跨越大洋的金属桥梁，不仅带来了物理介质，更带来了惊人的数字艺术革命。

## ShipIt 计划：非洲的铁盒与包邮的疯狂

2005 年前后，Canonical 公司启动了一个前无古人、且后来几乎再罕有来者的举措——**ShipIt（寄送光盘）计划**。

规则极其简单：只要你在网页上用邮箱注册并填入你的家庭或学校地址——无论是位于纽约繁华的曼哈顿，还是非洲草原深处的肯尼亚村落，或是中国某所不知名的县级中学——Canonical 均承诺免费打包、甚至免除一切国际昂贵邮费，将制作精良的 Ubuntu 安装光盘寄送到你家门口。

如果是对于那些校园社团的技术布道者或计算机教师，Canonical 甚至会在一个巨大的纸箱中，免费塞入多达哪怕百十张刻录光盘连同极具设计感的 Ubuntu 贴纸一并发货。你无需支付一美分。

从商业成本的算计上看，这是一桩极其荒唐的赔本生意。包装、光盘介质加上跨越半个地球的庞大国际快递与海关物流成本，在短短几年间消耗了马克高达数千万美元的私人财富。但从系统生态拓展与社区布道者的心理层面看，这又是一步神来之笔。

这不仅仅是一种对贫穷极客毫无居高临下之感的施舍，更是一种带有宗教热忱的馈赠。许多年轻人在收到人生中第一件来自海外的精美包裹时，那种对“自由分享精神”的触动与感激，化作了一粒粒种子。正是这些无数手捧光盘四处为同学“装机”的年轻人，硬生生地用人肉网络构筑了 Ubuntu 最为忠诚也是最具流量的早期社区基本盘。

没有 ShipIt 带来的草根狂奔，Ubuntu 就无法在后来动辄几千万甚至上亿装机量的 Linux 发行版混战中拔得头筹。

## 流光溢彩的棕色岁月：从 GNOME 2 到 3D 桌面狂欢

获得了受众并不等于留住了用户。在那个“颜即正义”的千禧年初期视听大爆炸年代，操作系统的 UI（用户界面）设计决定着它的命运。微软拿出了犹如蓝天白云般清爽的 Windows XP 及其华丽的各种视觉升级版，甚至后来的流线型透明 Aero 窗口（Windows Vista）。

而此时的 Ubuntu，却出人意料地走出了一条独具非洲人文审美的道路。马克决定摒弃当时大部分 Linux 偏爱的冷淡蓝色，为系统定下了极其鲜明的棕色与橙黄色基调。这种温暖、厚重、像广袤非洲红土地般色彩的图标设计，配合那些根据南非动物名精心命名的代号（如精明的小獾、飞扬的红头海雕等），让这套原本由冰冷 C 语言写就的内核，第一次具有了呼吸般的温度与人格。

伴随 2006 年 6.06 (Dapper Drake) 等 LTS（长期支持版）的发布，Ubuntu 开始步入了其最辉煌巅峰的桌面黄金时代。那时候的大多数桌面老派极客，都深切地迷恋着当时最为稳定高效的 **GNOME 2** 桌面环境。 Canonical 在这套桌面基础上不断地进行本土化微调、修复了无数恼人的中文字体渲染与驱动问题。

而真正让这个原本仅在极客圈流传的系统开始彻底破圈、甚至让无数用着 Windows 的非计算机专业大学生惊叹不已的，是极具代表性的 **Compiz（及其分支 Beryl）3D 桌面特效**。

很难用今天的平淡语言去描述，当一个从未见识过操作系统底层图形加速威力的普通人，第一次在这个完全免费的棕色桌面上开启 Compiz 特效时的那种震撼。你不仅能把四个虚拟桌面变成一个透明且可以任意旋转的 3D 立方体来回切换；还能在拖拽窗口时，让极其顺滑的窗口边缘如同真实的橡胶一般产生弹性扭曲扭动；当鼠标划过屏幕或者关闭程序界面时，窗口不仅会像真火一样熊熊燃烧殆尽，甚至还会在屏幕底部泛起一圈圈极其逼真的水波纹涟漪。

如果说，林纳斯证明了 Linux 在技术内核的效率上是无敌的，那么在 2008 年前后那段满屏幕炫酷 3D 的棕色岁月中，Ubuntu 则彻底地证明了：只要有一群有追求的极客在打磨一款系统，开源不但不丑陋，它甚至能美得傲慢，美得令同时期的所有闭源商业霸主流泪汗颜。

然而，正当所有人都沉浸在这段蜜月期，认为这款充满人情味的桌面系统必将一步步取代 Windows，引领全人类迈向属于开源的桌面极乐圣地时，那个坐在 Canonical 后台的男人，却敏锐地嗅到了时代齿轮错位摩擦的焦糊味。一场即将彻底撕裂开源社区神经元的大崩溃与野心，正在暗夜中疯狂酝酿。
