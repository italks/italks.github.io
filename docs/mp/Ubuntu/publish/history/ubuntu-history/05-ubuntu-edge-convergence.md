# Ubuntu 演进史（五）：失败的“大一统”革命——千万美元众筹“Ubuntu Edge”与移动端折戟

在科技产业的历史长河中，有很多极具前瞻性的产品，它们并非死于创意平庸，而是倒在了生态壁垒那冷酷的坚壁清野之下。

当我们回溯 Canonical 强推 Unity 界面、引发无数社区骂战与撕裂的动机时，如果仅仅将其视为一次狂妄的审美独裁，那显然是对马克·沙特尔沃思战略眼光的严重低估。Unity 背后隐藏着一个在当时极具科幻色彩、堪称宏大的技术乌托邦——**大一统（Convergence）**。

在 2013 年前后，智能手机的算力正在经历着指数级的狂飙。苹果和谷歌分别在手机和 PC 端维护着两套截然不同的操作系统（iOS/macOS 与 Android/ChromeOS）。而马克的野心是：为什么我们不能用一整套底层的操作系统架构，无缝地运行在手机、平板电脑、智能电视以及传统的桌面电脑上？

这就是 Canonical 在随后几年耗尽几乎全部心血的终极武器：**Ubuntu Touch**（及其底层的 Unity 8 界面和 Mir 显示服务器）。

## 疯狂的众筹：Ubuntu Edge 与 3200 万美元的豪赌

马克的构想是极度超前的。他设想的未来场景是：普通用户出门时，口袋里只需装着一部运行 Ubuntu 的超级智能手机。在路上，它有针对触摸屏优化的流畅界面；而当你回到办公室，只需把它插上底座，连接显示器、鼠标和键盘，它便瞬间无缝切换为拥有完整 Unity 界面的桌面个人电脑。

为了向全世界证明这个跨端融合理念的可行性，2013 年 7 月 22 日，Canonical 做出了一项在整个商业互联网众筹史上堪称疯狂的举动。

他们在著名众筹平台 Indiegogo 上发起了一个名叫“**Ubuntu Edge**”的超级智能手机硬件项目。这款充满未来主义、宛如一块纯黑金属切割而成、外观冷峻的设备，不仅在硬件配置上秒杀当时所有的旗舰机皇（拥有当时极度奢华的 128GB 存储、4GB 内存和蓝宝石玻璃屏幕），更是直接宣称出厂即搭载能实现无缝桌面切换的 Ubuntu Touch 与 Android 双系统。

这不仅仅是对发烧友的一次硬件售卖，这分明是一份投向整个智能终端产业链的战书。为了启动这台梦幻设备的大规模量产，Canonical 设置了一个极其令常人瞠目结舌的众筹目标金额：**3200 万美元**，并且期限仅有短短的 30 天。如果不能达到全额目标，所有支持者的款项将被全口径退回，项目也随之流产。

就在项目上线的第一天，极其恐怖的极客信仰瞬间引爆，仅仅 24 小时便狂揽了惊人的 340 万美元，打破了当时的众筹速度世界纪录。甚至连全球最大的财经新闻媒体——彭博社（Bloomberg），都为了支持企业的高可用办公，一举砸下 8 万美元订购了百台企业套装。

然而，奇迹只维持到了中段。在经历了狂热的技术极客早鸟票洗礼后，面对高昂的单机售价（初期 600 美元随后涨至 830 美元），普通消费者的热情迅速退烧。当 30 天的倒计时钟声在午夜敲响时，这个极具诗意与狂热的数字图腾，其众筹金定格在了 **1280 万美元**——尽管这同样创造了当时的全球众筹绝对金额之最，但距离那不可逾越的 3200 万美元目标，依然有着难以企及的天堑。

按照契约，所有资金被原路退回，Ubuntu Edge 手机在物理世界中胎死腹中。这是对 Canonical 资金链极其冷酷的一次拷问，也是马克那不可一世的桌面霸主梦想遭受的第一次真正意义上的重创。

## 冰冷的生态壁垒与最后的悲歌

尽管自研手机铩羽而归，但马克并未放弃他横跨移动端的“大一统”战略。在随后的几年里，Canonical 通过授权合作的方式，联合了来自中国的魅族（Meizu）科技和欧洲的 BQ 公司，艰难地推出了数款搭载 Ubuntu Touch 的量产型智能手机。

但是，从操作系统的商业演变史来看，一个系统哪怕拥有着再优雅的底层核心（哪怕那可是战无不胜的 Linux），如果它无法在那个充满泥泞的移动应用“集市”里找到足够厚实的泥土，它终究只能是一具华丽的骨架。

在这个阶段的世界上，智能手机的天空已经被 iOS 的围墙花园和 Android 那庞大的开放式联盟死死地锁死了太阳。当你拿着一部没有原生微信、没有 WhatsApp、玩不了甚至最简单的网络手游，连查个地图都必须使用网页版浏览器的 Ubuntu 手机时，那种“随时能变身成一台 PC”的技术浪漫，在日常社交的急迫与便利性面前，显得如此苍白无力且滑稽。

那些为了这个宏大愿景而在 Canonical 公司没日没夜重写了几百万行底层代码的顶级黑客们惊恐地发现，他们耗尽三年光阴构建的大一统城堡（Unity 8 与 Mir 显示协议），居然变成了一道在生态孤岛上自娱自乐的海市蜃楼。

在长达数年缺乏任何一家主流移动应用开发商跟进投资、公司营收遭到严重拖累的绝望中，无论是商业逻辑还是马克个人的财力，都逼迫着他必须做出那个哪怕是极其屈辱的断臂之举。一场宣告梦想破灭的悲壮大撤退，已经不可避免。
