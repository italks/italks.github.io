# Ubuntu 演进史（四）：痛苦的决裂——强推 Unity 桌面与初见端倪的商业化阵痛

从人类工程学演进的角度来看，极少有企业在拥有极高满意度与庞大信徒的巅峰期，敢于亲手砸烂那个为自己带来无数拥趸的殿堂。但在波诡云谲的科技赛道上，若是只盯着已有的蛋糕孤芳自赏，往往下一秒就会被海啸拍死在历史的沙滩上。

时间拨到 2010 年底。在这个阶段，Ubuntu 在 Linux 个人桌面上无疑是绝对的霸主，经典的 GNOME 2 桌面系统与那些华丽的 3D 特效几乎成为了所有极客的标配。但作为掌舵者的马克·沙特尔沃思，眼光早已越过了个人的方寸显示器。

此时的大背景是：乔布斯刚刚用 iPhone 与随后发布的 iPad 彻底颠覆了全世界关于“计算终端”的定义；而谷歌更是用基于 Linux 内核的安卓系统开始向全球移动端铺洒收割的大网。个人电脑（PC）出货量不可逆转地急剧萎缩；上网本（Netbook）和那些用手指进行触摸交互的长方形玻璃板正在成为新的图腾。

如果继续抱着那套只能用鼠标精准点击微小菜单的古典桌面系统，Linux 在消费级端必然走向衰亡。然而，他的下一次豪赌，却由于违背了开源的“集市”妥协精神，导致了 Canonical 历史上最为惊悚且痛苦的一次决裂。

## 拥抱未来的野兽：Unity 界面的降临

2011 年 4 月，随着 Ubuntu 11.04（代号“敏捷的独角鲸” Natty Narwhal）的发布，马克做出了一个震惊整个开源界，乃至在后来引起了近乎暴乱般反抗的决定：Canonical 毫无妥协地抛弃了极其成熟稳定且受人爱戴的传统 GNOME 界面，强行在系统中全局搭载了完全由自家独立研发的新型桌面环境——**Unity**。

Unity 这个词，本质上暴露了极大野望的冰山一角。从设计的底层逻辑看，这是一套完全专为触摸屏幕、宽屏全高清显示器，以及那些便携但纵向空间极低的上网本量身定制的交互哲学。

它不仅把所有程序的菜单栏和标题栏极端霸道地合并压缩融合到了屏幕最上方；更是将过去大家习惯放置在底部的任务栏硬生生挪走，变成了一根横亘在屏幕左侧、有着极其巨大且丑陋方块图标的无视鼠标滚轮习惯的全局启动器（Launcher）；甚至强行把最核心的操作枢纽融合进了一个类似于平板电脑搜索栏一样的全屏破网状面板（Dash）。

马克试图用这只野兽，硬生生把依然习惯于用鼠标工作的人类拉入未来的触屏大一统世界。

然而，他的激进显然彻底高估了极客们的忍耐力。

由于底层优化不佳，初代的 Unity 不仅极其卡顿、漏洞百出，甚至频繁导致系统因为桌面底层冲突而直接黑屏死机。对于那些刚刚习惯了由于各种微调配置而极其满意的老用户而言，这不仅是一次愚蠢的设计事故，更是一次极其残酷的“大教堂精英式”独裁对“开源集市社区民主”的践踏背叛。

抗议的邮件如同雪片一般挤爆了各大开源邮件列表论坛。原本和谐无比的社区瞬间撕裂，大量愤怒的高手选择直接倒戈跳船出走，去支持那些以传统桌面体验为卖点的衍生版（如 Linux Mint）。而马克对此展现出了极其硅谷式的霸道，不仅在技术决策上拒绝回头，甚至公开讽刺那些固守旧习惯的人。

但这只是决裂的开始，更加致命的信任危机还在于那不可触碰的道德红线。

## 挣脱乌托邦的商业化泥沼与隐私的污点

这不仅仅是技术之争或审美代沟。到了 2012 年 Ubuntu 12.10 发布时，一款原本清纯且不沾染铜臭味的操作系统，竟然开始为了钱而与用户产生冲突。

在之前的几年中，极具理想主义色彩的持续重金输血与每年按时发放的新功能维护服务器，已经花掉了马克个人以及 Canonical 公司巨额的现金流，且并未带来桌面端明显的增值反哺。公司不仅承受着巨大的财报压力，也必须为了那更为宏大的未来寻找造血机器。

于是，不可思议的一幕出现了：在 Ubuntu 全新的 Unity Dash（这个汇聚了用户最核心的本地寻找文件的搜索面板）中，用户的每一次本地文件搜索动作，系统居然会在后台悄悄通过网络，向亚马逊（Amazon）的商业 API 接口发送搜索请求，并在右下角极其突兀地展示出挂着电商推广链接的商品广告。每当有人点击购买，Canonical 就能借此抽取可怜的少量推广佣金提成。

在这个极其注重隐私与自由底色的开源世界，这就是名副其实的触犯天条。

不仅在广大普通社区中掀起了抵制的惊涛骇浪，“自由软件宗师”理查德·斯托曼（Richard Stallman）更是亲自下场，在博客上发出极其严厉的指责文章，直接将由于开启此项功能而违背“用户代理安全”的 Ubuntu 定性为“无耻的间谍软件（Spyware）”，呼吁所有的自由意志拥护者不仅要在安装首日彻底将其删除，更应该如同躲避瘟疫一般去摒弃它。

陷入四面楚歌的 Canonical 不得不在随后的一年中以修改系统默认关闭的形式退让，但在这种近乎惨烈的道德失血中，Ubuntu 丢掉了那层笼罩在极客心中完美的乌班图圣光。

这或许是所有成长中的开源巨头，甚至是整个现代数字工业从极客的诗与远方，跌进残酷资本下水道时必须忍受的切肤之痛与成人礼。然而，即便是承受了如此惨重的反噬，那颗名为 Unity 的种子背后所隐藏的终极计划——“大一统（Convergence）”，依然像一艘燃烧着、极速旋转失控的巨舰般冲向那个不可预知的未来设备战场，而那里，有着比这猛烈百倍的毁灭式风暴在等待着他。
