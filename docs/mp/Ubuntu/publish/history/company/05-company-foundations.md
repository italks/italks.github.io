# 开源巨头列传（五）：法杖与保护伞——FSF 与 Linux 基金会的幕后圣战

在波澜壮阔的开源演进大潮中，假如台前的聚光灯总是聚焦在像 IBM 或微软这样极其庞大耀眼的资本修罗场，或者林纳斯·托瓦兹（Linus Torvalds）这样极其狂躁的天才极客身上；那么我们很容易忽略在整个生态的最幽深不可见处，维系着这条跨越数十年的代码星河不被外界乃至内部力量撕裂的，其实是两套隐藏在暮色之下的“护法系统”。

一套是由理查德·斯托曼（Richard Stallman）所执掌的那把极其执拗清高的信仰法杖——**自由软件基金会（FSF）**；而另一套，则是汇聚了全球财阀极其庸俗的金钱以及交叉火力构成、专为守护极客不受侵扰免遭核杀的超级防弹衣——**Linux 基金会（Linux Foundation）及其衍生网**。在这两股截然不同的力量博弈与互补下，开源世界才没崩塌于人性的极度贪婪。

## 清教徒的法杖：FSF 与不妥协的圣殿

将时间闪回到了 20 世纪那个商业闭源犹如铁壁般不可一世的 80 年代，自由软件基金会（FSF）的建立，从第一天起就注定带着一种殉道般的苦行僧悲剧底色色彩。在斯托曼这位被无数圈内人敬如先知但同时又视为偏执狂精神怪咖的眼中，软件绝对不能有哪怕一丝的私有化妥协。

在这位极度抗拒一切“不纯洁剥削金钱”和“非自由限制”妥协的掌舵人带领下。FSF 极其强悍地为当时还是一盘烂骨烂沙处于雏形期的各种开源黑客，敲下并草拟了那份被后人无数次神化同时也畏惧的一纸大宪章：**通用公共许可（GPL，又名 Copyleft）**。

在当时乃至今天复杂的商业法院里，这份东西有着极为诡异和极具渗透杀伤的“病毒式极强传染性特质”：不论你是身价万亿的某个军工大集团乃至互联网霸主垄断者。如果你极其无耻地拿走哪怕借用了他们这一方在 GPL 受力下庇护着写出来的公开代码，只要这部分系统在世界公开流传发配乃至散布。你都将被强制极其残忍地立刻“全盘扒光内衣”，不仅不能闭门私藏，必须将你自己改良后的相关极其核心底层修改同样置于绝对开源且透明化的阳光暴晒中。

这曾让所有的商业贪婪财团如碰到了高位烈焰般在几十年的跨度中胆战心惊，也是在那些最极寒交迫乃至巨无霸极力垄断压榨绞杀时期内。能够硬生生强行捍卫那些底层底层弱小者发光、将整个“类 Unix 以及 GNU 重生基础底层核心生态大军代码碎片”保全不被吞没殆尽的最重要一根魔法定海神针。

## 庸俗且强大的防空洞：Linux 基金会的包围网

然而，极其吊诡的是；单纯且带有极度道德完美洁癖的信仰圣战，往往极度难以让那些真正极其庞大重工业级别的系统走入俗世，尤其是一旦遭到大企业那如同秃鹫般的法务狙杀以及动辄上百亿法庭纠纷核讹诈（例如 SCO 公司诉讼劫）的生死一线之际。

当到了林纳斯创造并极速狂飙让 Linux 系统登基成为足以动摇各个大公司极底盘乃至吞噬世界算力规模的时候。由于底层参与的开源开发公司实在太过庞大错综交横、每天进出上百上千名不可确定的各种杂乱外码汇集并进合并；你指望依靠一位天天极度愤怒且甚至拿不出一名世界顶尖商用重磅大律师的一群草台清流，去挡住财界毒蛇？

此时代表着最为务实且财大气粗的最粗防爆底线大本营——**Linux 基金会**（其前身为极其早期的开放实验室源等各种合并组建）以及极具交叉垄断护卫火力属性的 **OIN（Open Invention Network 开源发明共享网络机制）** 登场了。

林纳斯并不像斯托曼具有那种为了形而上的纯粹抗拒一切商业的狂人情结；这也是这两派直到今天甚至在如何命名这个极其庞大的内核系统时，在私下暗搓搓依然还在分裂暗斗指责的重要心结之一。但不能不承认这反而是全人类最为幸存的一幕。林纳斯选择了接纳那个以收下各种想要寄存在这套天下共主系统的商界财阀保护费建立的一个绝度强大基金安全阀！

这群由包括英特尔以及当时各种最巨型的跨国车企乃至金融机构巨无霸在极其恐怖且大资本重金联手拼盘资助下的所谓“天下开源最高联席大会”，彻底解绝了那个极其要死命保全的最棘手软肋。他们不仅让曾经天天在家里拿着微薄大学教员工资甚至不敢花钱的那个芬兰第一技术神明暴君（林纳斯本人）彻底衣食无忧，能够不再处理任何极其繁累的琐碎文书诉讼；更是联手砸下各种天文数字的安保护城河底盘资产抵押以及极其庞大的各类私密专利代码储备箱相互免费发誓交换、在不相互伤害对冲底线上签下了哪怕微软都不敢在这核聚变重围中挑事的天条戒碑协议。

这就是这长达极其惊心动魄且跌宕三十余年的开源地下双头怪的真面貌：用最为冷酷且甚至极其清冷固执的法杖在灵魂与意识顶楼守住了属于数字人性的基本尊严防止腐烂下沉；随后任由这套在红尘世俗世界里用重金、各种互相牵制以及各种大企业妥协交织出来的防弹财库金刚钟罩强硬托住了来自法制资本世界暗器的袭击射杀。两者互不可少，生生保住了这座极其宏伟且被各种惊雷闪电疯狂淬炼数载的最无敌之城！
