# 开源巨头列传（三）：蓝色巨人 IBM——十亿美元的世纪豪赌与抵御核弹的防波堤

如果要评选一部横跨科技产业百年兴衰的商业史诗剧，IBM（万国商业机器公司）无疑是这出戏里始终占据核心王座的“教父级”玩家。

在 20 世纪的大部分时间里，这家被世人敬畏地称为“蓝色巨人”的超级财阀，其商业逻辑一直是极其冰冷和封闭的。他们从最底层的硅片、大型机硬件（Mainframe），到中间件乃至操作系统（如自带的 AIX 等），实行的是如同封建领主般严密且利润丰厚的“垂直闭环收租”模式。在他们眼中，试图用几个免费命令替代数百万美金服务器的极客，无异于蚍蜉撼树。

然而，当历史的巨轮驶入 1999 年的时空节点，恰恰是这个最保守的闭源贵族，亲手挥舞起了砸向旧世界城墙的第一尊骇人雷神之锤。

## 逆向思维：蓝色巨人的十亿美元世纪宣战

千禧年前夕的服务器市场，正处于一种极其压抑的军阀混战状态。除了 IBM，还有 Sun Microsystems（拥有不可一世的 Solaris 系统）、惠普等巨鳄，再加上在桌面端称王正死死向数据中心狂咬的微软 Windows NT 系统。各家的高昂硬件与专有系统深度捆绑，让全世界的客户怨声载道。

当时的 IBM 掌门人郭士纳（Louis Gerstner）与随后接班的彭明盛（Sam Palmisano）拥有着超越了整整一个时代的极其骇人的战略嗅觉。

他们极其精准地判定了一件事：如果操作系统继续被这几家各自为战的私有系统割裂，那么不仅自己的市场会被微软慢慢蚕食乃至绞杀，而且这个世界永远不会迎来那个即将爆发的互联时代。要想在下一盘大棋中赢家通吃，最狠的招数就是“把操作系统这个底座生生做成不值钱的大众免费商品”。只有当各家的墙都倒了，IBM 最擅长的企业级 IT 咨询和高端服务器硬件才能卖得最疯。

2000 年的一天，《华尔街日报》的头条让整个科技界倒吸了一口冷气。IBM 以极其高调霸气的姿态向全世界宣布：他们将正式、全面地拥抱这只被众多友商唾弃的开源企鹅 Linux，并在随后更是砸下骇人听闻的 **10 亿美元（这是一个在当时可以买下数十家中型上市公司的天文数字）** 用于投入 Linux 全栈的技术研发、社区代码回馈与商业测试实验。

这不仅是极其血腥的降维打击。当刻板、西装革履的蓝色巨人将几百名全天下最优秀的底层系统程序员安插进 Linux 社区，开始夜以继日地为这个底层系统提交各种针对大型高可用服务器优化的顶配代码补丁时，他们实际上向全世界的五百强发出了一个极具威严的强力金字背书：你们不用再怕惹上麻烦，我们 IBM 用百年信誉作保，Linux 就是未来！

## 对抗 SCO 专利讹诈：极客与百亿大鳄的血色护城河

不过，这种用开源掀场子的暴走行为，很快也迎来了极其险恶的反扑。

2003 年，旧时代的落魄王孙 SCO Group 公司，由于无法忍受自己手里那张残缺的古典 Unix 版权变成废纸，对带头大哥 IBM 发起了一场足以写进法学教科书的专利核讹诈诉讼。他们极其恶毒地控诉 IBM 在拥抱 Linux 时“极其无耻地窃取并将 Unix 的绝密版权混入了 Linux 的代码中”，并开出了一个足以把中小型国企罚破产的天价索赔金额：**50 亿美元**。

并且由于 SCO 公司向全天下几千家企业滥发法律恐吓信，在当时瞬间引发了可能导致整个未成熟的开源界瞬间被摧毁的极大恐慌。这不仅是要杀 IBM，这是要把整个“集市里的极客”全盘用法律大棒钉死在耻辱柱上。

就在所有资本家都认为开源界会分崩离析的关键劫难时刻，历史见证了一场“西装暴徒”与“骇客狂徒”最完美的奇幻结盟。

IBM 极度高傲地拒绝了一切庭外妥协与花钱消灾的想法。他们调集了全美最顶级的王牌百年法务战备精英团（这群拿着数千美元时薪的高智商怪物甚至比懂代码的人还懂专利链），与从那全世界各个暗网角落中自发为了保卫 GPL 协约而集结起来的无数底层硬核极客（如著名的 Groklaw 分析论坛），形成了一道不可逾越的护城长城墙。

在长达十数年的拉锯拉练中，IBM 法务就像用微雕刻刀一样，配合着数以万计的黑客逐行找出的代码开源铁证前序发源历史邮件，在一场又一场的法庭辩论上将 SCO 提出的所有所谓证据不仅全部击碎，更在最后直接翻底牌，极具戏剧性地向法官证明了 SCO 连最初声称这块专利完全归他的最核心资格都存在致命欺诈和买卖硬伤。

## 防波堤的建立

这场耗尽心血的世纪反诉最终以原告 SCO 悲惨的濒临破产变卖清算作为全戏的大结局。这也是整个自由开源界第一次真刀真枪地在那个极其充斥着尔虞我诈的专利丛林体系中扛下了极其恶毒的核攻击。

IBM 在这场劫难中不仅彻底向全球买家证明了：跟着我用 Linux，就算天塌下来也有我的精英给你扛雷。更是在随后的岁月里，主导结成了诸如开放发明网络（OIN）这样互换专利的核互保联盟。如果说那群开源人（如 Linus）是修建底层神庙的设计师，那么在这数十年间毫不手软地挥舞钞能力及百年法务镰刀清扫一切荆棘的蓝色巨人，无疑是庇护这座神庙免遭战火蹂躏的最强“持剑守陵人”。
