# Linux 演进史（七）：知识产权的生死保卫战——SCO 诉讼案与 Linux 基金会

从商业史上看，所有颠覆旧秩序的新兴事物，在成长为能够左右行业格局的参天大树之前，必然会遭受既得利益集团的一场残酷围剿。在 21 世纪的第一个十年，Linux 虽然在服务器领域凭借极致的高可用性和开放的社区生态攻城拔寨，但它始终头顶悬挂着一把极其锋利的达摩克利斯之剑——知识产权。

开源代码犹如汪洋大海，每天都有来自全球成千上万个账户的数百万行匿名代码汇入。这在客观上必然会导致对底层专利边界的极度模糊。对于传统软件巨头而言，他们无法在技术战场上用代码击溃 Linux，唯一的杀招，便是在极其险恶的“法庭角斗场”中发起知识产权诉讼，以制造恐慌来阻断这股狂潮。

2003 年爆发的 SCO 群体诉讼案，正是这一暗黑战术达到巅峰状态的最凶险时刻。

## 50 亿美元的讹诈：专利流氓的全面反扑

在整个操作系统的古典纪元，UNIX 是当之无愧的霸主。它的原始版权几经转手，其中的核心代码最终流落到了一家名叫 SCO Group 的公司手中。当时的 SCO 公司在市场竞争中节节败退，几乎濒临破产边缘，但这并不妨碍他们手中的 UNIX 知识产权成为一种极度危险的战略核武器。

2003 年 3 月 6 日，SCO 公司做出了一个震惊整个硅谷的举动。他们一纸诉状将投资 Linux 最为激进的科技巨头 IBM 告上了法庭，直接索赔骇人听闻的 10 亿美元（随后更是丧心病狂地修改诉状，将金额飙升至 50 亿美元）。SCO 的指控其心诛毒：他们声称 IBM 在共同开发项目中“极其严重地窃取并抄袭了”UNIX 系统中最宝贵的商业机密代码，并非法地将其捐献并混入了 Linux 的底层内核中。

更为致命的是，SCO 并没有停下他们法律抹黑的脚步。几个月后，他们将枪口直接对准了广泛使用或者发售 Linux 的商业企业（例如 Red Hat 和 AutoZone 等），甚至狂妄地向全球 1500 家世界 500 强企业群发了恐吓信。信中赤裸裸地威胁道：只要你们的服务器里还在运行带有所谓“窃取代码”的 Linux 操作系统，那么等待你们的将是极其严重的侵权连带责任甚至天价罚款，除非你们按照每个 CPU 高达数百美元的价格向 SCO 乖乖缴纳“授权保护费”。

这是极其高水平的核讹诈，也是开源史上最黑暗的时刻。

本质上，大型跨国企业对操作系统的技术指标虽然苛刻，但在动辄上亿利润的法律风险评估面前极度保守。面对 SCO 这种专利流氓的无赖搅局，大量原本准备转向 Linux 阵营的企业陷入了严重的恐慌与观望，那些依赖 Linux 生态生存的公司股价也应声暴雨般连阴。整个刚经历过互联网泡沫的开源产业不仅遭遇了严重的信任危机，它那种无中心化的“集市模式”，在法庭上连一个拥有雄厚财力作为辩护人的主体都找不到。

对于当时还极其稚嫩的开源法务边界来说，一场生存还是毁灭的大保卫战，被迫打响了。

## “不妥协的洁癖”与开源共同体的长城抗战

当遭遇极其阴狠的司法绞杀时，人类组织有时不仅需要资本家的高谈阔论，更需要那些有着“道德洁癖”的布道者挺身而出。

理查德·斯托曼以及他缔造的那份充斥着理想主义色彩的 GPL（通用公共许可证）在这个危难时刻显示出了极其强大的抗脆弱性。由于 GPL 从一开始就将自由分发与对窃取专利的零容忍写入了基因中，它使得面对这种泼脏水的行为，全球的开源黑客不仅没有分崩离析，反而如同被侵夺家园的狂信徒一般，自发组织了一次空前绝后的“网络人民战争”。

无数工程师停下了手头的开发工作，在各大极客社区（如 Groklaw 论坛）的组织下，他们开始逐行逐句、甚至逐个字符地对比从 20 世纪 70 年代第一版 UNIX 代码至如今 Linux 2.4 内核之间几百万行的海量差异。他们挖空心思地翻出那些被遗忘十余年的技术邮件与会议记录。

最终在法庭上，IBM 的精英律师团与背后无数无名的黑客力量形成了绝佳的合力。每一次 SCO 声称某段有争议的代码来自于他们时，开源社区都能迅速指出那些代码要么早己被公开共享，要么其实是由前人甚至林纳斯重写的实现逻辑。更具戏剧性的是，由于产权链条的倒卖过于频繁，真正保留原有 UNIX 专利的公司其实是 Novell，SCO 连起诉的最基本资格都不具备！

经过长达近十年的诉讼缠斗，这场被称为开源界特洛伊木马的连环诉讼不仅以 SCO 公司的全面败诉并破产清算而告终，它甚至连一丝能够撕裂 Linux 核心生态的缝隙都没有敲开。

## 历史的防波堤：Linux 基金会与 OIN 的诞生

从历史演进的周期律看，每一次极度惨烈的劫局，往往会倒逼出更为稳固的基础设施。SCO 诉讼案给了硅谷巨头及所有开源信守者一次极其沉痛的教训——在丛林法则盛行的知识产权专利战中，开源界绝不能再如以前那样是一盘毫无法律防备的散沙。

因此，为了构筑一条应对专利流氓的战壕，一系列专注于给林纳斯进行法律与体系保护的实体组织如雨后春笋般出现。在这之中，最具代表性的便是 Linux 基金会（Linux Foundation）的前身，以及由各大科技巨头联手组建的“开放发明网络”（Open Invention Network，简称 OIN）。

OIN 堪称开源世界的核保护伞。包括 IBM、谷歌、索尼乃至于后来的微软在内，他们将数以万计的底层核心专利捐献出来打包，并定下一条极其铁血的规矩：“任何想要免费使用我们这个防波堤里的庞大专利库的人，唯一的前提条件就是必须签署互不侵犯条约——承诺永不用自己手中的专利去起诉 Linux 系统以及任何核心开源组件的合法开发者。”

这在本质上将几十家相互搏杀、甚至有着世仇的大型科技财阀，强行在 Linux 这个底座之上捆绑成了利益共同体。林纳斯本人也得以摆脱繁杂的诉讼与杂务指控，由 Linux 基金会高薪全职供养，让他得以静下心来专注他的技术内核更新。

经此一役，Linux 虽然遭受了极其严重的名誉及司法重创，却也在这场血雨腥风中淬火重生，真正奠定了长达数十载无人敢予撼动的金刚不坏之体。而当历史的指针即将拨入下一个智能时代时，再也没有任何法律或是对手，能够阻挡这只企鹅向全人类的数字版图进行最后的吞噬了。
