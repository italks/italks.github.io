# 传奇人物传（三）：马克·沙特尔沃思 —— 将星辰赐予平民的太空过客

> “软件本身应该是免费的，其价值在于它能为人类创造什么。” —— 马克·沙特尔沃思

在开源世界的群星里，马克·沙特尔沃思（Mark Shuttleworth）是一个极其特殊的坐标。他既不是斯托曼那种苦行僧般的道德圣徒，也不是林纳斯那种纯粹的技术疯子。他更像是一位文艺复兴时期的赞助人，或者是一个试图用资本杠杆撬动科技公平的骑士。

如果套用吴军《计算之魂》的逻辑，马克·沙特尔沃思解决的是计算机科学中最难的一个环节：如何让精英手中的复杂工具，变成每一个普通人都能握住的权杖。

## 从数字证书到星辰大海

在创立 Ubuntu 之前，马克已经是一个成功的商业传奇。他在南非的自家车库里创立了 Thawte，由于准确捕捉到了电子商务早期对数字证书的需求，他最终以 5.75 亿美元的价格将公司卖给了 Verisign。

对于大多数亿万富翁来说，接下来的生活可能是游艇与高尔夫。但马克却做出了两件足以证明他异于常人格局的事情：
1. 他花费 2000 万美元成为了人类历史上第二位自费太空游客，在国际空间站俯瞰了这颗蓝色的星球。
2. 他决定用剩下的钱，去解决地面上那个日益严重的“数字鸿沟”问题。

那是一个属于 Internet Explorer 垄断桌面、Linux 依然是只有极客才能驾驭的字符终端的年代。马克意识到，如果没有一个能让老奶奶也会用的图形化 Linux 系统，开源软件的自由理想就永远只是少数人的精英主义自嗨。

## Ubuntu：对他人仁慈的哲学实现

2004 年，一个代号为“疣猪临时工（Warty Warthog）”的项目诞生了，这就是后来的 Ubuntu。这个词源于非洲祖鲁语，意为“我的存在是因为大家的存在”，或者通俗地翻译为“人性”。

马克在 Ubuntu 上展现出了极高的工程管理与产品意识。他不再纠结于内核的微小改动，而是专注于“用户体验”。他提出了著名的“ShipIt”计划：无论你身处地球的哪个角落，只要你申请，Canonical 公式都会为你免费寄送 Ubuntu 的系统安装光盘，甚至连邮费也由他们全包。

这种看似疯狂的撒钱模式，打破了软件分发的地理壁垒。成千上万来自非洲、亚洲和南美洲的孩子，正是通过这些漂洋过海的光盘，第一次接触到了能与 Windows 媲美的先进操作系统，并由此改变了命运。这在本质上，是将计算机科学的边界，从硅谷实验室推到了人类文明的最边缘。

## 挫败与转型：从理想主义桌面到云端霸主

在《计算之魂》中，商业逻辑与技术实现的冲突是永恒的主题。马克曾试图推行大一统（Convergence）战略，想要让手机、平板和桌面跑在同一个 Ubuntu 系统上。但在 iOS 和 Android 的降维打击下，这场战役遭遇了惨败。

然而，马克的伟大之处在于他的韧性与敏锐。当他意识到桌面市场的堡垒难以攻破时，他迅速带领 Ubuntu 转身拥抱云计算和 AI。如今，当你使用 AWS、Azure 或是运行最新的大语言模型时，很有可能它们背后运行的底层系统正是 Ubuntu。

马克用他的智慧向世人证明：开源并不是商业的对手，而是商业的更高维度。

## 结语

在去往太空的旅程中，马克曾说他看到地球是如此美丽而脆弱。这种从宇宙视角带来的悲悯心，最终具象化为了 Ubuntu 每一张光盘、每一个半年度更新的版本。

他是一个用商业手段实现理想主义的异数。他告诉世界，计算机科学的终极价值，除了代码的简洁与算法的正交，还有一份跨越种族与财富的重千钧的“人文关怀”。在开源史的大航海时代，他就是那个不仅自己上船，还要带全人类去看星星的人。
