# 开源巨头列传（四）：微软帝国——从“万圣节绞杀”到“俯首称臣”的魔幻反转

在整整两代计算机工程师的心中，如果要在科技史上挑选一个比《星球大战》中达斯·维达更具有绝对压迫感和反派光环的角色，那一定是在20世纪末由比尔·盖茨和史蒂夫·鲍尔默所统治的那个不可一世的**微软帝国（Microsoft）**。

在长达几十年的时间里，微软和开源阵营（特别是 Linux 操作系统周边生态）之间的仇恨，绝不仅仅是简单的市场份额之争，那几乎是一场关乎信仰、价值观与底层商业逻辑生死存亡的“极权与叛军”的宗派残酷圣战。在微软极其封闭且严密的“售卖版权收租”的帝国大厦看来，以彻底免费且具备极强传染性（GPL）自居的 Linux 企鹅，就是盘旋在华盛顿州雷德蒙德总部的挥之不去的极度恶毒的梦魇。

## “万圣节文件”与血色绞肉机

为了捍卫 Windows 以及其 Office 等各种绑定的千亿闭源霸权，世纪之交时期的微软展现出了令人胆寒的獠牙。

1998 年的前夜，开源界精神领袖埃里克·雷蒙德收到了一份随后引爆全球科技圈内幕档案，这段丑闻后来被人称作“**万圣节文件（Halloween Documents）**”。这一系列严重泄密的微软内部绝命备忘录极其坦诚且刺骨地暴露出：傲慢的微软并非是不懂技术，他们甚至比任何人都要清楚这套去中心化研发的开源内核，在长期演进后不仅其底代码质量已经优于自己闭门造车的产物，而且其不计成本收割服务器市场的姿态“极其深重地威胁着微软基于收费代码赖以为生的基本盘”。

为了将其扼杀于摇篮，微软动用了一切见不得光的超级暗器。他们不仅在公司内部把具有免费扩散属性的 GPL 许可证定性为极其难听且骇人的“商业毒瘤（Cancer）”，并且开始隐晦地策动被称为“**拥抱、扩展然后再绞杀融合（Embrace, Extend, and Extinguish）**”的连环毒计：故意在公开的网页和文件传输协议底层强塞某些只能被自家 IE 浏览器甚至 Windows 环境私下识别的不兼容恶心小插件，用极其恶毒的技术墙壁生生地逼迫用户无法离开他们的环境。在后台，他们更是隐匿身形，甚至暗中默许和资助类似于 SCO 种类的法务流氓疯狂状告 Linux 用户群体。

那段时间，对阵微软帝国的 Linux 极客们，就如同在一场永远见不到反败光明的地下极寒游击战中苦苦厮杀。

## 失落的十年与无可挽回的双重塌陷

然而，极其吊诡的历史周期律永远令人生畏。你永远无法在技术革命前夜，用旧时代的霸权马鞭去阻挡哪怕最难看但最狂野的新型汽车的降临。

在长达十年的拉锯中，微软在 PC 桌面端确实如日中天不可逼视，但由于其庞大身躯对收费闭源路径的极其执念固执，他们付出了极其惨痛的高傲代价。在2010年代全球突然从桌面切换到随身手机这块疯狂大蛋糕时。在底层完全采用 Linux 以及各种开源组合拼装且极其“白嫖免费”的 Android （安卓）系统和苹果 iOS 夹击下，微软极度笨拙甚至妄图再次收取系统授权金的的手机端（Windows Phone 系统）被这场生态降维群殴打得粉身碎骨，血本无归。

而在那支撑他们最大后方利润的云端服务器市场。面对成千上万个轻薄容器沙盒化、每天在虚拟机里生灭数万次的云计算浪头。昂贵、庞大、甚至每次加载都拖泥带水的各种闭源大型操作系统，在一众极其精巧、免费甚至根本都不用重启就完成集群调配升级的服务器版 Ubuntu 甚至 CentOS 大军中，更是被抢走了全球极其恐怖的云计算虚拟节点主基调份额。

帝国的统治地盘在除了 PC 家庭端之外的几乎所有边缘世界和云端算力机房遭到了血洗。

## “微软爱 Linux”与惊世骇俗的 WSL 木马融合

当生死存亡悬于一线时，只有拥有绝高智慧的实干家才能挥刀自宫，逆天改命。

接替那些只会狂吠打压开源前任的新掌舵者，这位于2014年力挽狂澜临危受命的萨提亚·纳德拉（Satya Nadella），是一位极其清醒且拥有冷酷“大算力思维”云原生改革家。他一上任，立刻看透了在这个以公有云 Azure 为利润生死线的新世界中，任何和全球占主导的后台算力系统——Linux 开发生态作对的行为，无异于带着全公司极其愚蠢地跳崖自杀。

在一场让全球老一代极客差点惊掉下巴微软大会上。曾经最仇恨企鹅的巨人背景屏幕上，极其魔幻且显眼地打出了一行字：**“微软爱 Linux (Microsoft Loves Linux)”**。

这不仅仅是一句恶心的公关词汇。纳德拉用极其狠辣的手段强行叫停了和开源社区所有的版权嘴炮与司法围剿，并加入了旨在互换专利和绝不起诉开源的极客安全防波堤。

但他最神来之笔的招数，是向全球的程序员极其妖孽地发布了那个名为 **“Windows 子系统（WSL - Windows Subsystem for Linux）”** 的核级大招。微软极其舍得血本，在其内部利用最高权级的硬件调度虚拟机制虚拟底层，强行与诸如 Canonical （Ubuntu 的开发商）甚至是各种红帽分销商达成极其亲密合作定制协议，让你可以直接在满布花哨壁纸、不仅能够打着 3A 大作游戏甚至挂着办公套件的 Windows 的底层“原生跑出了一个甚至连文件系统盘符都能无缝映射的纯粹开源企鹅骨肉”。

这是一出极其令人战术叹服的“相生相克”无间道。表面看，微软是对着他追杀了二十年的死敌低下了极其高贵的头颅，用这种近似投降般承认失败的手段，默许了开源系统在各种代码开发端的无可匹敌乃至绝对优势；但从底牌战略上看，这是纳德拉的一手极尽借体还魂的奇招：既然我在云端底层和极客号召力上打不过你，那我就极其恶心地“物理融合你”。当你所有的开发、运维环境和各种跑着黑框框的各种高纬网络编程工作都能借由这个“体内木马”顺畅完成时。也就意味着全球上千万极其高端核心骨干程序员极客，终于再也不必因为极其难以折腾的不死机环境而去忍痛离开微软的操作系统了界面了！

从那被鲜血染红的无尽缠斗乃至恶毒绞杀，最终走向一个我中有你、并且在全球共同为了极大型计算提供最底层的狂飙算力的极度深度交融。微软与开源（Linux）这场长达四分之一个世纪乃至魔幻色彩的旷世之战，注定会成为人类算力工业文化碰撞史上最具极其深远张力的经典兵法阳谋。
