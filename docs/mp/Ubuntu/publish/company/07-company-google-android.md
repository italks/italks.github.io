# 开源巨头列传（七）：谷歌安卓——收割万亿果实的隐形寄生帝国

在开源软件长达半个世纪的波澜演进中，无论是以桌面化体验为傲的 Ubuntu （Canonical），还是在企业级服务器机房敲响印钞机的红帽（Red Hat），他们本质上都还在遵循着最为传统的“计算机”外形的路径依赖中厮杀。

但如果有一天，你从大历史的终局视角去俯瞰这场战争，你会极其错愕地发现：真正把 Linux 这只冰寒地带的企鹅，毫不在意地塞进这个星球上几十亿人类从老到幼的裤兜里、手腕上、甚至甚至是汽车座舱乃至家电里的，既不是林纳斯本人，也不是以上任何一家纯血脉的传统极客霸主。

而是一家靠在网页框里做关键字搜索起家、在二十一世纪初前十年以无可匹敌的碾压之势，将开源理念化为最极致的一场“寄生与变异”移动端突围战役的恐怖异刃——**Google（谷歌与 Android 安卓帝国体系）**。

## 绕开“死结”的神来之笔：Bionic 与 HAL 架构

时间回到 2005 年。当时的智能手机市场被称作“带软盘的移动砖头”，那是塞班（Symbian）、黑莓（BlackBerry）以及微软那极其笨拙昂贵的 Windows Mobile 割据的旧石器群岛。当谷歌前瞻性地用极其便宜的价格收购了由极客安迪·鲁宾（Andy Rubin）捣鼓出的开源手机底层系统 Android（安卓）时。他们面对的，是一个看似无解的局。

当时的谷歌极其清楚：如果想要在最短时间内干掉微软和诺基亚的闭源系统，唯一能拿出手的极其强大、免费且多任务神级核心只有 Linux 内核。

但是这里横亘着一个极其要命的死罪红线。Linux 采用的是理查德·斯托曼那极其严苛、且像剧毒病毒一样强制共享开源的 **GPL 许可证**。这就是说：如果哪家手机芯片或者硬件厂商（比如高通、三星乃至各种极其自私封闭的代工厂）敢基于这套系统写了一套极其私有赚钱发财的硬件驱动器（比如摄像头或者基带网卡调频），那么他们也必须被迫将这些商业机密级别的底层驱动代码全部裸露且彻底开源！

这简直就是要了老式硬件垄断大厂的命！在那种环境下，如果有这层“原教旨主义的极客道德代码锁”，全天下没有任何一家商业财阀敢给谷歌做安卓的手机。

就在此时，谷歌的大脑层做出了整个计算历史上堪称“变异级别”也最具争议神魔性质的操作。为了既吸食享用 Linux 极品免费底层的强大并发力量，又要极其冷酷安抚资本家厂商的私有时光。谷歌团队用极其浩大的精妙工程力：生生用 BSD 等宽容执照新写了名为 **Bionic libc** 的替换库，更是在半空中极其强硬架设了一层将系统内核与上方业务斩断隔开的 **HAL（硬件抽象隔离隔离层）**！

这招叫“借尸还魂”：在最深的硬件层以下，那些极其核心的驱动秘密不仅不再需要直接和极其烫手的 GPL 核心死死相连被迫释放污染代码！只要厂商向上用闭源且包在外壳极其私密的 API 传输；再配合最上面全是完全免费随便改哪怕闭源代码售卖甚至私有占用的 Apache 协议的大面积“非 Linux 传染库”！这只看似开源极其博爱的安卓，实际上是被谷歌极其精算大手术改造过、可以极其完美套用隐藏在无数私有壁垒硬件下还能逃离开源大棒的神级缝合大怪兽！

## 恐怖的暴兵大潮与隐形霸权

这个近乎钻了开源协议极其惊险且致命漏洞的代码空间魔法（尽管此后至今依然遭到诸多自由基金会老学究破口大骂其实伪开源甚至极其投机）。不仅彻底且疯狂地让高通、博通这些老牌吸血硬件乃至天下几乎中国所有的极低利润手机草莽代工组装机器厂彻底毫无顾忌胆怯了包袱！

不仅免费，而且保你商业独占这不被人开源泄密！这种不可抵挡大降维生态，不仅在接下来十年化作一场连微软鲍尔默砸破脑袋都想不明白这恐怖的海啸！甚至在极短极快直接彻底生生冲垮踏烂了整个旧王朝（不可一世塞班帝国陨落乃至诺基亚王国破产）。也让这套挂着极小一部分绿色代码企鹅芯，却披挂着甚至让厂商极大自主权连带满眼塞着谷歌巨大框架的大机器：不仅收割了占世界极绝大份的人口屏！更是极其无情打碎了让任何妄想向桌面收取版权系统暴发户绝望溃败的大门槛。

## “农村包围城市”的极高终局大一统

而今时日。那些极客大儒所坚守极其极其神圣的大版图；在这家搜索引擎巨擘极具魔改借力的降维打击前已经变成了隐性的背后面孔。

你甚至会在那些普通大众玩着抖音刷着极低智网剧中。谁会在意这极其冰冷绝命流畅的后台进程、其多维线程甚至是那处理几十亿极其复杂指令的最极深沟里那一头大大的 Linux 大企鹅正在呼吐吞云？

那曾是无数像 Ubuntu （马克）极其痴迷但折戟沉沙甚至惨败千万美金都没能跨越将世界屏幕融为一体的大宏大之梦（Convergence）。却被极其精算不按套路但极其狡猾凶猛的谷歌搜索帝国，甚至连姓名都没宣扬就暗度陈仓、以极其霸道近乎在硬件和各种白嫖框架极尽压榨包夹狂飙的手腕下：以极其强悍不留情“农村包围城市”把属于极其硬核最强内核霸权生生地、连根拔起种满了世界各个哪怕最不经意的电器插头深处。

这就是这长达跨越极其半世纪从开源极至最后大财阀集极至魔合，最冷酷也但确是最为丰硕万亿极其极其登峰造极收割的史诗级大完结闭环剧幕。
