# Rufus 4.12 发布：启动盘制作神器的又一次进化 + 新手保姆级教程

> 作为 Windows 上最轻量、最靠谱的 USB 启动盘制作工具，Rufus 刚刚迎来了 4.12 版本的更新。这次更新不仅修复了“开发驱动器(Dev Drive)”的识别问题，还增强了安全性。本文将带你了解新版变化，并手把手教你制作一个万能系统启动盘。

## Rufus 4.12 更新了什么？

Rufus 4.12 虽然版本号跳跃不大，但在稳定性和安全性上做出了重要改进，特别是针对开发者和高级用户。

### 1. 智能识别改进：告别误操作
在 Windows 11 中，微软引入了 **Dev Drive (开发驱动器)**，这是一种基于 ReFS 的高性能存储卷。之前的版本可能会错误地将这些虚拟驱动器列为可移动设备，导致用户误格式化。
**Rufus 4.12 完美修复了这个问题**，它现在能智能区分物理 USB 设备和虚拟开发驱动器，大大降低了“手滑”删库的风险。

### 2. ISO 兼容性增强
你是否遇到过因为 ISO 镜像路径太长，或者文件名包含特殊字符而导致 Rufus 报错？
新版本重构了 ISO 解析引擎，增强了对**非标准 ISO 结构**和**长文件路径**的支持。无论你是折腾冷门的 Linux 发行版，还是制作定制化的 Windows 镜像，成功率都得到了显著提升。

### 3. 安全性升级 (TOCTOU)
这是一个底层的安全修复。Rufus 4.12 修复了一个 **TOCTOU (Time-of-Check Time-of-Use)** 漏洞。简单来说，这意味着当 Rufus 以管理员权限运行时，恶意软件更难利用它来篡改系统文件。虽然普通用户感知不强，但这让它在企业环境中更加安全可靠。

## 保姆级教程：如何制作 U 盘启动盘？

无论你是要重装 Windows，还是想体验最新的 Ubuntu Linux，制作一个启动盘是第一步。

### 准备工作
1.  **一个 U 盘**：容量建议 **8GB 以上**（Windows 10/11 镜像通常超过 5GB）。
    *   *注意：制作过程会清空 U 盘所有数据，请提前备份！*
2.  **系统镜像 (ISO)**：从微软官网或 Linux 发行版官网下载。
3.  **Rufus**：从官网 (rufus.ie) 下载最新版（推荐便携版 Portable，无需安装，即点即用）。

### 制作步骤

#### 第一步：运行 Rufus 并选择设备
插入 U 盘，双击运行 Rufus。
*   **设备**：Rufus 通常会自动识别你的 U 盘。如果有多个，请仔细核对盘符（如 `E:` 或 `F:`）。

#### 第二步：选择镜像
点击 **“选择 (SELECT)”** 按钮，找到你下载好的 ISO 镜像文件（例如 `ubuntu-24.04-desktop-amd64.iso` 或 `Windows11.iso`）。

#### 第三步：关键设置（MBR 还是 GPT？）
导入镜像后，Rufus 会自动调整设置，但有一项需要注意：
*   **分区类型 (Partition scheme)**：
    *   **GPT**：**推荐**。适用于绝大多数近 5 年购买的电脑（UEFI 启动）。
    *   **MBR**：仅适用于非常古老的老爷机（Legacy BIOS 启动）。
    *   *小技巧：如果不确定，选 GPT 通常没错。如果启动不了，再回来重做成 MBR。*

#### 第四步：开始制作
点击底部的 **“开始 (START)”** 按钮。

*   **如果是 Windows 镜像**：Rufus 会弹出一个“用户体验”窗口，允许你：
    *   跳过微软账户登录（创建本地账户）。
    *   跳过 TPM 2.0 检测（让老电脑也能装 Win11）。
    *   *按需勾选，非常方便！*

*   **如果是 Linux 镜像**：如果提示检测到 ISOHybrid，直接默认选择 **“以 ISO 镜像模式写入”** 即可。

#### 第五步：等待完成
点击确定后，Rufus 会开始写入。根据 U 盘速度，通常需要 3-10 分钟。当进度条变成全绿，并显示 **“准备就绪 (READY)”** 时，就可以拔掉 U 盘去装系统了！

## 为什么我们依然推荐 Rufus？

市面上有 Ventoy 等“拖入即用”的工具，为什么 Rufus 依然是装机必备？
1.  **极致的兼容性**：Ventoy 偶尔会遇到某些主板或奇葩 ISO 无法启动的情况，而 Rufus 采用的是最传统的解压/引导方式，**成功率接近 100%**。
2.  **Windows 定制能力**：能够一键屏蔽 TPM 检测、强制创建本地账户，这是装机员的最爱。
3.  **开源免费无广告**：体积仅 1MB 出头，没有任何后台残留。
