# GNOME 50 "东京"来了：纯 Wayland、游戏帧率暴涨20%，Ubuntu 26.04 桌面大揭秘

昨天 Ubuntu 26.04 Beta 开放下载，很多朋友第一时间尝了鲜。

但装完之后，不少人问我同一个问题：

**"这桌面怎么和 24.04 感觉不太一样了？"**

答案很简单——因为 Ubuntu 26.04 搭载了全新的 **GNOME 50**。

GNOME 50（代号 **"Tokyo"**）于3月18日正式发布，距离上个版本刚好半年。这一次的更新，不是小修小补，而是一次**底层架构级的跃迁**。

今天这篇文章，带你把 GNOME 50 的每个重要变化都看一遍。

---

## 一、最大的变化：纯 Wayland 时代正式到来

如果你对 Linux 桌面有一定了解，一定听过 Wayland 和 X11 之争。

GNOME 50 做了一个果断的决定：**彻底移除 X11 会话支持**。

这意味着什么？

### 从今以后，你只能用 Wayland 登录 GNOME

以前在登录界面可以选择 "GNOME" 或 "GNOME on Xorg"，现在 Xorg 选项**直接消失了**。

GNOME 团队认为 Wayland 经过多年打磨，已经足够成熟，可以全面接管。而且 Wayland 在安全性、多显示器支持、HiDPI 适配等方面确实比 X11 优秀得多。

### 旧应用怎么办？

别担心——Wayland 本身兼容运行 X11 应用。你常用的那些基于 X11 的老旧软件，依然可以正常运行，GNOME 通过 **XWayland** 兼容层提供向后支持。

真正受影响的，是那些依赖 X11 独有特性（比如全局快捷键注入、屏幕录制工具）的软件。好消息是，主流应用基本都已完成 Wayland 适配。

### 对 Ubuntu 26.04 的影响

Ubuntu 26.04 作为 LTS 版本，会随 GNOME 50 一同进入纯 Wayland 时代。Canonical 一直在推动 Wayland 生态，这次 GNOME 的决定与其战略完全一致。

**一句话总结：Wayland 是未来，X11 正式成为历史。**

## 二、游戏玩家注意：N 卡帧率暴涨，VRR 全面优化

如果你在 Ubuntu 上玩游戏，GNOME 50 是一个值得兴奋的版本。

### NVIDIA 游戏性能大幅提升

GNOME 50 针对 NVIDIA 显卡做了多项底层优化：

- **修复画面卡顿问题**：引入底层规避方案，解决之前 Wayland 下常见的帧生成异常
- **默认开启可变刷新率（VRR）**：支持 FreeSync / G-Sync 的显示器现在开箱即用，无需额外配置
- **超分功能默认启用**：在更多硬件配置下自动开启，提升画面清晰度
- **低延迟光标**：开启 VRR 时，鼠标光标渲染独立于游戏帧率，保持丝滑响应

### 实测效果

综合目前的测试数据，搭载 GNOME 50 的 Ubuntu 26.04 在游戏性能方面大约可以提升 **10% 到 20%**，主要体现在帧率稳定性上。

在部分使用 GravityMark 等基准测试工具的场景中，也能看到类似的提升。

### HDR 屏幕共享

还有一个容易被忽略但很实用的改进：**分享屏幕时可以完整保留 HDR 元数据**。

以前通过 OBS 或其他工具共享屏幕时，HDR 内容会发灰、色彩失真。GNOME 50 引入了 Wayland v2 色彩管理协议，分享 HDR 内容时色彩准确度大幅提升。

这对游戏直播和远程协作场景来说是个好消息。

## 三、远程桌面：Vulkan 硬件加速，延迟大降

GNOME 的远程桌面功能（GNOME Remote Desktop）在 50 版本中获得了**史诗级升级**。

### Vulkan + VA-API 硬件加速

这是本次更新最硬核的改进之一：

- 利用本地 **GPU** 进行视频流编码
- 引入 **Vulkan** 和 **VA-API** 硬件加速
- 大幅降低远程桌面延迟和功耗

简单来说，以前远程桌面的视频编码靠 CPU 软编码，现在交给 GPU 硬件编码了，**延迟更低、画质更好、CPU 占用更小**。

### 显式同步（Explicit Sync）

NVIDIA 显卡用户在远程桌面中经常遇到画面撕裂或卡顿。GNOME 50 引入了显式同步支持，为 NVIDIA 用户带来了**更稳定的画面表现**。

### 企业级功能

- **HiDPI 支持**：自动完成远程桌面的缩放适配
- **摄像头重定向**：远程会话中可以直接调用本地网络摄像头
- **Kerberos 认证**：为企业级应用提供安全保障

## 四、家长控制：从"没有"到"全方位"

这是 GNOME 50 中**最出人意料的亮点功能**。

### 之前：几乎没有

说实话，之前的 GNOME 在家长控制方面几乎是空白。Linux 桌面虽然安全，但如果孩子拿到电脑，基本上想怎么玩就怎么玩。

### 现在：全面上线

GNOME 50 直接把家长控制做到了**系统底层**：

| 功能 | 说明 |
|:---|:---|
| **屏幕时间管理** | 为儿童账号设置每日使用时长限制 |
| **就寝计划** | 设定"到点锁屏"，到时间自动锁定 |
| **灵活延长** | 到时间后可以给额外时长（家长批准） |
| **网页过滤** | 系统底层内置网页过滤，未来通过界面更新即可强制内容过滤 |
| **专属应用** | 全新设计的"家长控制"设置面板 |

### 对 Ubuntu 26.04 的意义

Canonical 一直把 Ubuntu 定位为"全家人都能用的操作系统"。GNOME 50 的家长控制功能，填补了 Linux 桌面在家庭场景下的最后一块短板。

**以后给孩子用 Ubuntu，终于不用提心吊胆了。**

## 五、无障碍体验：更包容的数字世界

GNOME 50 在无障碍方面也做了大量工作。

### Orca 屏幕阅读器全面升级

Orca 是 GNOME 的屏幕阅读器，50 版本的变化非常全面：

- **全新偏好设置界面**：与 GNOME 整体设计语言统一
- **智能语言切换**：根据网页或应用界面语言自动切换播报语言
- **全局配置 + 应用定制**：可以设置全局默认，也能为特定应用定制专属配置
- **Wayland 全面支持**："鼠标复查"功能在 Wayland 下完全可用
- **盲文显示增强**：粘滞模式在 Electron 应用中自动开启

### "减少动态效果"开关

系统设置中新增了 **"减弱动态效果"** 选项。

开启后，界面动画会被精简，降低视觉干扰。这对容易因屏幕动画感到眩晕的用户来说，是一个非常贴心的改进。

## 六、核心应用更新

### 文件管理器：性能蜕变

Nautilus（文件管理器）在 GNOME 50 中获得了显著改进：

- **缩略图加载速度大幅提升**：打开图片密集的文件夹不再转圈等待
- **内存占用显著降低**：底层使用 Glycin 库进行安全沙盒化图像解码
- **批量重命名重构**：交互更直观，"替换文本"新增高亮提示
- **文件属性面板升级**：改为独立悬浮窗口，不再占用主界面
- **搜索增强**：支持叠加多个文件类型过滤器

### 文档查看器：批注功能彻底重构

批注系统从里到外重写了一遍：

- 点击主界面按钮即可直接添加批注
- 支持**文本输入、划线、内容高亮**
- 提供多种颜色、线条粗细选项和橡皮擦工具

这个改进让 PDF 批注体验接近了专业工具的水平。

### 日历应用：生产力跃升

- **新增参会者列表**：清晰查看"必须参加"和"可选参加"成员
- **快速添加事件**：重新设计，逻辑更清晰
- **事件导出**：支持导出为 ICS 文件
- **月视图优化**：月份名称更醒目，事件对齐更整齐

## 七、开发者关心的变化

### VRR + 分数缩放修复

如果你用分数缩放（比如 125% 或 150%），之前和 VRR 一起用可能有各种 Bug。GNOME 50 全面修复了这些问题。

### 色彩管理升级

Wayland 引入了 v2 版本色彩管理协议，确保专业显示器、HDR 屏幕的色彩精准还原。

这对设计师、摄影师等需要精确色彩管理的用户来说非常重要。

### 每周第一天设置

系统设置新增"每周第一天"选项，会全局同步到日历等应用。

虽然看似小事，但对中国用户来说，**设置周一为每周第一天**是个很实用的功能。

## 八、GNOME Circle：新应用生态

GNOME Circle 是 GNOME 官方的第三方应用项目，50 版本新增了几个值得关注的应用：

| 应用 | 功能 |
|:---|:---|
| **Gradia** | 屏幕截图精修与批注，支持渐变背景、阴影等效果 |
| **Constrict** | 智能视频压缩，支持设定目标大小和多种编码 |
| **Sessions** | 番茄工作法可视化计时器 |
| **Sudoku** | 现代化数独应用，极简设计，全键盘操作 |

## 九、Ubuntu 26.04 会怎么定制？

Canonical 历来会对 GNOME 做一些定制。基于历史规律和目前信息，预计 Ubuntu 26.04 会保留以下 GNOME 50 原生特性：

**大概率保留**：
- 纯 Wayland 桌面
- 游戏性能优化
- 远程桌面 Vulkan 加速
- 文件管理器改进
- 文档批注功能

**可能调整**：
- 家长控制可能简化或推迟（LTS 注重稳定性）
- 壁纸和主题会换成 Ubuntu 风格
- 部分新应用可能通过 Snap 分发

## 总结

GNOME 50 "东京"的核心关键词是：**成熟、性能、包容**。

- **成熟**：告别 X11，Wayland 全面接管
- **性能**：游戏帧率提升、远程桌面加速、文件管理器变快
- **包容**：家长控制、无障碍升级、更完善的国际化和本地化

作为 Ubuntu 26.04 的默认桌面，GNOME 50 让我们对这个 LTS 版本更有信心了。
