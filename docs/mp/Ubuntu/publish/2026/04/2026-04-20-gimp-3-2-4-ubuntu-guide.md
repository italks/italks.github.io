## GIMP 3.2.4 到底更新了什么？

### 🔧 核心改进：图层工作流大优化

本次 3.2.4 版本虽然是个维护更新，但修复的问题都是**日常高频痛点**：

#### 1. 操作适用性修正（告别误操作）

以前在 GIMP 中执行某些操作时，经常遇到"这个操作不应该对这种图层生效"的情况。现在规则明确了：

| 操作 | 适用范围 | 不再影响 |
|:---|:---|:---|
| 图层调整为图像大小 | 仅位图图层 | ❌ 矢量图层、文本图层、链接图层 |
| 调整图层以匹配选区 | 仅位图图层 | ❌ 非位图类型 |

这意味着你再也不用担心**一不小心把文字图层拉伸变形**了。

#### 2. 导入图层命名 Bug 修复

这是一个困扰很多用户的实际问题：

> **问题场景**：你在项目 A 中打开了一个 XCF 文件作为图层，结果导入的图层名字显示的是文件名而不是原始图层名。

**现在修复了！** 导入的图层会正确保留原始名称，团队协作时不再混乱。

#### 3. 其他稳定性改进

- **裁剪工具增强**：对非光栅图层的支持更好
- **按颜色选择工具**：交集模式下性能显著提升
- **选择工具**：移动浮动图层时自动禁用干扰项
- **PDF / PSD 导出**：兼容性和质量进一步提升

### 🎯 矢量图层（Vector Layers）

这是 GIMP 历史上**首次支持真正的矢量图层**！

```bash
# 类比理解：
```

GIMP 3.2 引入了更多**非破坏性编辑**能力：
- 裁剪不再永久删除像素（可恢复）

## Ubuntu 用户如何安装 GIMP 3.2.x

### 方法一：Flatpak（推荐）

Flatpak 提供最新的 GIMP 版本，且与系统隔离，稳定可靠：

```bash
# 安装 Flatpak（如果尚未安装）
sudo apt install flatpak flatpak-plugin-system

# 添加 Flathub 仓库
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

# 安装 GIMP
flatpak install flathub org.gimp.GIMP

# 启动 GIMP
flatpak run org.gimp.GIMP
```

**优点**：版本最新、自动更新、不影响系统包管理

### 方法二：Ubuntu 官方源（适合保守用户）

```bash
# Ubuntu 24.04 及以上
sudo apt update
sudo apt install gimp

# 查看版本
gimp --version
```

**注意**：官方源的版本可能稍旧（如 3.0.x），建议优先使用 Flatpak。

### 方法三：Snap

```bash
sudo snap install gimp
```

Snap 版本也较新，但启动速度比 Flatpak 慢一些。

| 维度 | Flatpak | Snap | APT 官方源 |
|:---|:---:|:---:|:---:|
| 版本新旧 | ⭐ 最新 | ⭐ 较新 | ⭐ 可能较旧 |
| 启动速度 | ⭐⭐ 快 | ⭐ 一般 | ⭐⭐⭐ 最快 |
| 更新频率 | ⭐ 及时 | ⭐ 及时 | ⭐ 跟随发行版 |
| 磁盘占用 | ⭐⭐ 约 500MB | ⭐⭐ 约 500MB | ⭐ 最小 |

**结论**：想要最新功能的用户选 **Flatpak**，追求稳定的用户选 **APT**。

## 实用技巧：让 GIMP 更好用的小设置

### 1. 界面优化（单窗口模式）

```
菜单栏 → 窗口 → 单窗口模式 (Ctrl+Shift+W)
```

这样 GIMP 看起来更像现代软件，不会到处浮动窗口。

### 2. 深色主题（护眼）

```
编辑 → 首选项 → 界面 → 主题 → 选择 "深色"
```

### 3. 常用快捷键速查

| 快捷键 | 功能 |
|:---|:---|
| `Ctrl+N` | 新建图像 |
| `Ctrl+O` | 打开图像 |
| `Ctrl+S` | 保存（XCF 格式） |
| `Ctrl+Shift+E` | 导出为 PNG/JPG |
| `B` | 画笔工具 |
| `U` | 模糊/锐化 |
| `Shift+G` | 渐变工具 |
| `[` / `]` | 调整画笔大小 |
| `Space` | 临时切换为拖动视图 |

### 4. 安装插件扩展功能

GIMP 支持丰富的插件生态（类似 Photoshop 的滤镜）：

```bash
# G'MIC（强大的滤镜集合）
sudo apt install gmic

# Resynthesizer（智能填充/去除水印）
sudo apt install gimp-plugin-registry
```
