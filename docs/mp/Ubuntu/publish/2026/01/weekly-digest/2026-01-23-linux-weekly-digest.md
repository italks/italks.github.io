## 🦄 头条：Ubuntu Core Desktop 26 开放预览

**"你的系统炸了？重启就好。"**

本周，Canonical 正式发布了 **Ubuntu Core Desktop 26** 的首个技术预览版（Technology Preview）。
与传统的 Ubuntu 不同，这是一个**"不可变系统" (Immutable OS)**。

*   **这是什么？** 整个根文件系统 (`/`) 是只读的。你不能随便修改系统文件，甚至不能用 `apt` 安装软件到系统层。
*   **怎么装软件？** 全部依赖 Snap、Flatpak 或 Ubuntu 自己的容器技术 (LXD/Docker)。
*   **有什么用？** 极致的稳定性。系统更新就像手机 OTA 一样，后台静默下载，重启生效。如果更新挂了？系统会自动回滚到上一个版本。

**💬 极客辣评：**
> 对于喜欢折腾配置文件的老鸟来说，这简直是噩梦；但对于不仅想修电脑、只想写代码的开发者来说，这可能是最完美的开发环境。

## 🐧 内核：Linux 6.14 或将移除对旧款 x86 硬件支持

在 Linux Kernel 邮件列表（LKML）中，一场关于"清理遗产"的讨论正在激烈进行。
维护者建议在即将到来的 **Linux 6.14** 中，彻底移除对 **非 ACPI (高级配置与电源接口)** x86 硬件的支持。

这意味着，如果你家里还有 2000 年以前的奔腾 II/III 甚至 486 级别的古董机器，想跑最新的 Linux 内核可能要变成奢望了。
Linus Torvalds 对此表示："We are not a museum."（我们不是博物馆）。

**⚡ 影响范围：** 99.99% 的现代用户不受影响，但复古计算爱好者可能要停留在 6.10 LTS 了。

## 🔥 GitHub 热榜：Atuin v18.0 —— Shell 历史记录的 AI 革命

你是否经常遇到这种情况：
*"我上周输入的那个超复杂的 `ffmpeg` 命令是什么来着？"*
然后疯狂按 `Up` 键或者 `Ctrl+R`？

**Atuin** (GitHub ⭐ 45k) 本周发布了 v18.0 大版本更新，正式引入了 **本地 LLM 语义搜索**。

*   **以前：** 你必须记得命令里的关键词。
*   **现在：** 你可以直接搜自然语言。
    *   输入：`search "如何压缩视频"`
    *   Atuin 自动匹配历史记录：`ffmpeg -i input.mp4 -vcodec libx265 -crf 28 output.mp4`

而且，这一切都发生在本地，不会上传你的隐私数据。这才是 2026 年该有的终端体验！

## 🛠️ 工具推荐：Zed 编辑器 Linux 版正式稳定

经过一年的 Beta 测试，号称"快过 VS Code，美过 Sublime"的 Rust 编辑器 **Zed**，终于宣布 **Linux 版进入 Stable 阶段**。

*   **渲染性能：** 基于 GPU 渲染，打字延迟低至 10ms 级。
*   **协作开发：** 内置类似 Google Docs 的多人实时协作。
*   **AI 融合：** 深度集成了 GitHub Copilot 和 **DeepSeek**（没错，支持自定义端点！）。

如果你厌倦了 Electron 应用的内存占用，Zed 值得一试。
`sudo snap install zed --classic`
