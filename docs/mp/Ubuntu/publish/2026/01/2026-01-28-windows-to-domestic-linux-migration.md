# 告别 C 盘焦虑？从 Windows 迁移到国产 Linux 的「生存指南」

> 🎯 **适用对象：** 刚接触统信 UOS、麒麟、Deepin 等国产操作系统的用户
> 💡 **核心痛点：** 软件怎么装？C 盘在哪里？原来的习惯还能用吗？
> 🏷️ **关键词：** 国产化替代, Linux 迁移, UOS, Deepin, 麒麟操作系统

“单位电脑换成了国产系统，完全不会用了怎么办？”

随着信创（信息技术应用创新）的推进，越来越多的办公电脑换上了国产 Linux 操作系统（如统信 UOS、银河麒麟、Deepin 等）。很多用惯了 Windows 的朋友面对这个“熟悉的陌生人”，第一反应往往是手足无措。

界面看着挺像 Windows，但点进去发现逻辑全变了。别慌，这篇文章就是为你准备的**差异化生存指南**，帮你快速理清那些“反直觉”的概念。

---

## 📂 差异一：我的 C 盘、D 盘去哪了？

**Windows 思维：**
电脑必须分 C、D、E、F 盘。系统装 C 盘，软件装 D 盘，资料存 E 盘。

**Linux 现实：**
**在 Linux 世界里，没有“盘符”的概念。**

Linux 的文件系统是一个树状结构，所有的文件都从根目录 `/` 开始。
*   **`/` (根目录)：** 相当于“我的电脑”。
*   **`/home/你的用户名`：** 这才是你的地盘！相当于 Windows 的 `C:\Users\用户名`。
    *   你的文档、下载、桌面文件全都在这里。
    *   **建议：** 以后存东西，直接扔进“文档”或“下载”文件夹即可，不用纠结是存在 C 盘还是 D 盘。Linux 的空间管理非常智能，你不需要像在 Windows 那样担心“C 盘爆红”。

> **🧠 观念转变：** 忘记分区的概念，把 `/home` 当作你的万能“D 盘”。

---

## 📦 差异二：软件安装，告别“全家桶”

**Windows 思维：**
打开浏览器 -> 百度搜索软件名 -> 下载 `.exe` -> 双击安装 -> 一路狂点“下一步”（顺便被捆绑安装了 3 个浏览器和 2 个杀毒软件）。

**Linux 现实：**
**应用商店是第一选择。**

国产 Linux 发行版通常都内置了体验非常好的**应用商店**。
1.  打开应用商店。
2.  搜索软件（微信、WPS、腾讯会议等）。
3.  点击“安装”。

**优势：**
*   **纯净：** 没有捆绑软件，没有弹窗广告。
*   **安全：** 都是官方审核过的版本，不会下到病毒。
*   **统一更新：** 系统更新时，软件也能顺便一起更新。

如果商店里没有，你可能会接触到 `.deb` 格式的安装包，双击它也能安装，但这通常是备选方案。

> **🧠 观念转变：** 找软件先去“商店”，而不是去“百度”。

---

## 🔌 差异三：驱动程序，鲁大师“失业”了

**Windows 思维：**
装完系统第一件事：下载鲁大师/驱动精灵，把显卡、声卡、网卡驱动扫一遍，不然屏幕分辨率不对，或者没声音。

**Linux 现实：**
**驱动集成在内核里，即插即用。**

Linux 的大部分驱动都包含在系统内核中。当你插上鼠标、键盘、甚至打印机时，系统通常会自动识别并加载驱动。你不需要手动去下载驱动程序。

**但也别高兴太早：**
如果是非常新的硬件（比如刚发布的显卡）或者非常老旧特殊的设备（老式扫描仪），Linux 的驱动支持可能不如 Windows 及时。但在国产化办公场景下的标准硬件，兼容性已经做得相当不错。

> **🧠 观念转变：** 只要设备能动，就不要试图去“更新驱动”。

---

## 📝 差异四：常用软件的“平替”与“原生”

这是迁移中最痛苦的部分，但情况正在好转。

| 软件类型 | Windows | 国产 Linux (UOS/Deepin/麒麟) | 体验差异 |
| :--- | :--- | :--- | :--- |
| **办公文档** | Microsoft Office | **WPS Office** (Linux 版) | 几乎无缝衔接，WPS 在 Linux 上表现极佳，且没有广告。 |
| **即时通讯** | 微信/QQ | **微信/QQ (Linux 原生版)** | 功能比 Windows 版少（比如可能没有朋友圈、小程序），但基本的聊天、传文件完全没问题。 |
| **浏览器** | Edge/Chrome | **自带浏览器** (通常基于 Chromium) | 内核一样，插件通用，书签可以同步。 |
| **作图** | Photoshop | **GIMP / 在线设计工具** | 这是一个痛点。虽然有 GIMP，但操作逻辑差异大。建议改用 Canva、搞定设计等在线工具，或使用国产的修图软件。 |
| **压缩** | WinRAR/Bandizip | **归档管理器** | 系统自带，支持 zip, rar, 7z，无需额外安装。 |

> **⚠️ 特别提醒：** 某些依赖 **IE 内核** 的老旧网银或 OA 系统，在 Linux 上可能会无法打开。这是因为 IE 技术已被淘汰，建议联系 IT 部门询问是否有非 IE 的访问方式。

---

## 🛡️ 差异五：关于“杀毒”与“权限”

**Windows 思维：**
必须装个 360 或火绒，不然心里不踏实。右下角经常弹出“由于您很久没清理垃圾...”。

**Linux 现实：**
**裸奔即可。**

*   **权限机制：** Linux 的权限管理非常严格。软件想修改系统文件？必须输入密码授权。病毒很难在后台悄悄获得最高权限。
*   **生态环境：** 针对 Linux 桌面的病毒本身就极少。

你会发现，没有了杀毒软件的后台扫描，同样配置的电脑，Linux 运行起来往往更流畅。

---

## 🚀 总结

从 Windows 到国产 Linux，**不是“降级”，而是一次“换血”**。

刚开始的一周你会觉得很别扭：
*   找不到 C 盘很慌；
*   不能右键刷新桌面（其实 Linux 不需要刷新）；
*   装软件没有“下一步”的快感。

但跨过这个适应期，你会爱上它的**纯净、专注和高效**。没有弹窗广告的打扰，没有强制更新的蓝屏，电脑回归了它最本质的作用——工具。
