## 二、OpenClaw + skills.sh：到底是什么？

### 1. OpenClaw是什么？

OpenClaw（GitHub 12万+星标）是一个开源、本地优先的AI智能体平台。它的核心定位是：**24小时在线的数字员工**。

与传统聊天机器人不同，OpenClaw能主动理解你的意图，并通过调用各种工具，自主执行跨平台的真实任务。你可以理解为：一个由大语言模型驱动的"数字员工"，就运行在你的电脑上。

它的使用方式也很简单——通过微信、钉钉、飞书这些日常聊天工具，用自然语言告诉它要做什么，它就帮你去执行。

### 2. skills.sh是什么？

如果说OpenClaw是"员工"，那skills.sh就是这个员工的"技能库"。

skills.sh是AI智能体技能目录（The Agent Skills Directory），定位类似于AI代理领域的"应用商店"或"npm包注册表"。上面汇集了各种可安装的AI技能，装上就能让AI具备特定能力。

安装方式也非常简单，一条命令搞定：

```bash
npx skillsadd <owner/repo>
```

比如安装PDF处理技能：

```bash
npx skillsadd pdf
```

### 3. clawhub.ai是什么？

这是另一个技能获取渠道——面向智能体的快速技能注册中心，核心特点是支持向量搜索，能帮你更精准地找到需要的技能。

## 四、安装实操：4步搞定

### 步骤1：环境准备

你需要准备以下环境：

- Node.js 18+
- npm 或 pnpm
- Mac/Windows/Linux 均可（推荐Mac mini，性价比高）

### 步骤2：安装OpenClaw CLI

```bash
npm install -g openclaw
```

### 步骤3：安装技能

以安装PDF处理技能为例：

```bash
npx skillsadd pdf
```

安装完成后，OpenClaw就具备了处理PDF的能力。

### 步骤4：验证安装

```bash
claw list
```

这个命令会列出所有已安装的技能，确认安装成功即可开始使用。

## 五、真实场景：装上技能后能做什么？

### 场景1：自动生成PPT周报

**痛点：** 每周要花2小时写工作总结PPT

**技能组合：** xlsx + pptx

**效果：** 把上周的工作数据导入Excel，告诉OpenClaw"帮我生成周报PPT"，它会自动读取数据、填充内容、格式化排版，5分钟搞定。

### 场景2：批量处理客户资料

**痛点：** 每周要整理上百份PDF合同，提取关键信息

**技能组合：** pdf + docx

**效果：** 把PDF合同丢给OpenClaw，它会自动提取甲方名称、合同金额、到期日期等关键信息，整理成结构化的Word文档。

### 场景3：自动化测试报告

**痛点：** 每次发版要做UI测试，耗时且重复

**技能组合：** playwright + xlsx

## 六、避坑指南：这些坑别踩

### 坑1：技能版本不兼容

**表现：** 安装后技能无法使用，报错提示版本不匹配

**解决：** 在skills.sh页面查看技能支持的OpenClaw版本，确保你的OpenClaw版本在支持范围内。命令 `claw --version` 可以查看当前版本。

### 坑2：权限配置错误

**表现：** 技能提示无法访问文件或浏览器

**解决：** 检查 `~/.claw/config.json` 中的权限配置，确保已授权技能所需的目录访问权限、浏览器自动化权限等。

### 坑3：技能依赖冲突

**表现：** 同时安装多个技能后，某个技能报错

**解决：** 建议逐个安装技能，每装一个验证一个。优先安装官方认证的技能（带verified标识）。
