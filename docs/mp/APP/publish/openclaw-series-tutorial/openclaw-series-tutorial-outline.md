# OpenClaw 系列教程大纲规划

> **定位**：面向小白用户，从零开始系统学习 OpenClaw，实现从"不知道是什么"到"能用好、用得深"的完整进阶路径。
>
> **目标读者**：对 AI 自动化感兴趣的普通人、职场白领、小团队负责人，无需编程基础。

---

## 系列定位与理念

### 教学理念

1. **循序渐进**：每篇文章只解决一个问题，难度递增
2. **场景驱动**：不讲空洞概念，用真实场景引出功能
3. **可操作性**：每个步骤都有截图和命令，照着做就能成功
4. **避坑导向**：提前预告常见问题，减少踩坑

### 系列结构

```
入门篇（是什么、能干什么）
    ↓
基础篇（怎么用）
    ↓
进阶篇（如何用好）
    ↓
实战篇（真实场景应用）
    ↓
高级篇（深度定制与扩展）
```

## 写作规范建议

### 统一格式

1. **标题**：主标题 + 核心问题
2. **开头**：一个场景或痛点引入
3. **正文**：小标题 + 步骤 + 截图
4. **结尾**：本篇总结 + 下篇预告
5. **附录**：相关资源链接

### 语言风格

1. 口语化但不随意
2. 多用比喻解释技术概念
3. 每个操作都有截图
4. 预告可能的坑并提供解决方案
