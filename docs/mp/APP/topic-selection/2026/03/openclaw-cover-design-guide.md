# OpenClaw 系列教程微信公众号封面图设计方案

## 封面图规格要求

- **尺寸**:900 x 383 像素(微信公众号首图文封面)
- **格式**:JPG 或 PNG
- **风格**:统一、专业、简洁

---

## 统一设计元素

### 品牌色系
- **主色调**:#0066CC(科技蓝)
- **辅助色**:#FF6600(活力橙)
- **背景色**:#F5F7FA(浅灰)
- **文字色**:#333333(深灰)

### 字体规范
- **标题**:思源黑体 Bold / 阿里巴巴普惠体 Bold
- **副标题**:思源黑体 Regular
- **数字**:DIN Pro / Roboto

### 图标风格
- 扁平化设计
- 线条简洁
- 统一圆角

## 各阶段封面设计

### 第一阶段:入门篇(3篇)

#### 第1篇:OpenClaw 是什么?3 分钟看懂 AI 自动化新物种

**设计主题**:初识 OpenClaw

**视觉元素**:
- 中央:OpenClaw Logo + 问号图标
- 左侧:传统办公场景(堆积的文件)
- 右侧:AI 助手图标(机器人或智能体形象)
- 背景:渐变色(从灰到蓝)

**文字内容**:
```
主标题:OpenClaw 是什么?
副标题:3分钟看懂 AI 自动化新物种
标签:#入门篇 01/19
```

**设计说明**:用对比手法,左侧展示传统办公的繁琐,右侧展示AI自动化的便捷,中间用OpenClaw Logo连接。

## 制作建议

### 推荐工具

1. **稿定设计**:中文模板丰富,适合快速制作
2. **Canva**:国际知名,设计元素多
3. **创客贴**:国内工具,操作简单
4. **Figma**:专业设计工具,适合团队协作

### 设计流程

1. 选择合适的模板(科技类、教程类)
2. 替换品牌色和字体
3. 添加核心视觉元素
4. 调整文字内容
5. 导出为JPG/PNG

### 注意事项

- 保持统一风格,增强系列感
- 文字简洁有力,突出重点
- 颜色对比度适中,确保可读性
- 图标清晰,避免过度装饰
- 预留Logo和二维码位置

---

## 封面图系列感设计

为了让19篇文章形成系列感,建议:

1. **统一边框**:所有封面左侧添加彩色边条,按阶段变色
   - 入门篇:绿色
   - 基础篇:蓝色
   - 进阶篇:橙色
   - 实战篇:紫色
   - 高级篇:红色

2. **统一Logo**:右上角统一放置OpenClaw Logo

3. **统一标签**:右下角添加"系列教程XX/19"标签

4. **统一字体**:全文使用相同字体系统

5. **统一风格**:扁平化、简洁、现代
