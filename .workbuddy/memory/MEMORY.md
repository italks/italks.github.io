#### 常见问题与解决方案
- **内容丢失**：`parseHtmlFile()`不要处理空白字符，保留原始HTML结构
- **文章合并**：每篇文章单独调用`addDraft()`API，传入单篇文章数组
- **内容过长+标签不闭合**：使用`truncateHtml()`智能截断
  - 在完整块级元素处截断（</p>, </div>, </h1-6>, </li>, </ul>, </ol>, </table>, </pre>, </blockquote>）
  - 自动统计并闭合未关闭的HTML标签
  - 添加截断提示框（橙色背景+居中文字）
  - 确保最终长度在20000字符限制内
  - ⚠️ **已暂停使用**：截断影响阅读体验
