## 🔄 多设备协作说明

### 核心原则
- **读取**：可以查看所有设备的工作记录
- **写入**：只修改当前设备的 memory 文件
- **同步**：通过 Git 同步所有设备的 memory

### 如何查看其他设备的工作记录
```bash
# 查看所有设备的最新记录
ls .workbuddy/memory/devices/*/2026-04-04.md

# 查看指定设备的记录
cat .workbuddy/memory/devices/desktop-pc/2026-04-04.md
```

请查看 `.workbuddy/memory/README.md`
